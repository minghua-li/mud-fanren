## 一、日常修行活动

### 1.1 打坐修炼（核心经验获取）

| 修炼方式 | 效率 | 消耗 | 适用场景 |
|---------|------|------|---------|
| 普通打坐 | 1x 基础速度 | 无 | 洞府/静室/野外 |
| 聚灵阵修炼 | 1.2-3x | 灵石/时辰 | 门派的聚灵室、私人洞府 |
| 灵脉修炼 | 2-5x | 需要灵脉地段 | 门派核心区域、占灵脉为洞府 |
| 丹药辅助修炼 | 1.5-4x | 丹药（一次性） | 配合任何修炼方式 |
| 双修 | 1.5x（双方） | 需双修功法 | 道侣之间 |
| 战斗修炼 | 0.5x（战斗同时） | 无 | 猎杀妖兽时兼得经验 |

**打坐机制：**
- 打坐消耗「游戏时间」，每时辰（现实 5 分钟）获得一次经验结算
- 灵根资质决定基础倍率（伪灵根 x0.3，真灵根 x1.0，天灵根 x2.5）
- 可离线挂机打坐（离线最多 8 小时，期间按正常效率的 60% 计算）
- 突发事件（妖兽袭击/天劫/同门遇险）可打断离线挂机

### 1.2 境界突破

| 突破阶段 | 前置条件 | 所需道具 | 成功率 | 失败后果 |
|---------|---------|---------|-------|---------|
| 炼气→筑基 | 功法 7 层+ | 筑基丹 | 10%（服药后） | 修为倒退 1-3 层，半年内无法再尝试 |
| 筑基→结丹 | 筑基后期大圆满 | 凝液丹/天火液 | 1%（无辅助） | 修为倒退至筑基中期，一年冷却 |
| 结丹→元婴 | 结丹后期大圆满 | 补天丹 | 极低 | 丹碎人亡（修为清空）或假婴（修为剩三成） |
| 元婴→化神 | 元婴后期大圆满 | 化神丹 | 几乎不可能 | 元婴受损，百年难以恢复 |

**游戏化设计：**
- 突破时进入独立副本「渡劫空间」，玩家需在空间内完成心魔挑战
- 心魔挑战 = 连续答对修仙感悟题/击败心魔幻象
- 突破成功 = 境界 + 寿元增加 + 解锁新技能
- 突破失败 = 修为倒退 + 冷却期 + 属性暂时下降

### 1.3 炼气期日常（新手阶段）

| 时间段 | 活动 | 收益 |
|-------|------|------|
| 早晨 | 做师门杂务（扫地/挑水/照料灵田） | 门派贡献 +1~3，少量经验 |
| 午后 | 听讲经/自学功法 | 功法熟练度 + 少量经验 |
| 下午 | 采集灵药/猎杀低阶妖兽 | 灵石 1-5，材料 |
| 傍晚 | 坊市闲逛/社交 | 信息获取，偶尔捡漏 |
| 夜间 | 打坐修炼 | 基础经验获取 |
| 随机 | 触发江湖/门派事件 | 额外奖励 |

**炼气期核心体验：** 让玩家感受修仙世界的真实感和「弱小时期的挣扎」。这是最长的游戏阶段之一，但不应该是枯燥的——通过频繁的小事件和渐进式成长维持兴趣。

### 1.4 筑基期日常（主力阶段）

| 时间周期 | 活动 | 收益 |
|---------|------|------|
| 每日 | 切磋论道/执行门派巡逻 | 实战经验 + 门派贡献 |
| 每周 | 猎妖小队（3-5 人组队） | 妖兽材料 + 灵石 |
| 每旬 | 坊市交易日 | 倒卖/淘宝 |
| 每月 | 门派任务结算 | 俸禄 + 贡献兑换 |
| 每季 | 秘境开放/门内小比 | 稀有产出 + 声望 |
| 不定期 | 追杀逃犯/剿灭匪修 | 额外赏金 |

**跨地图历练链（筑基期推荐路线）：**

```
越国境内（炼气→筑基初期）
  ↓ 筑基中期后可出越国
元武国/姜国/车骑国（筑基中期）
  ↓ 筑基后期
乱星海（筑基后期→结丹）
```

### 1.5 结丹期以上日常（高阶阶段）

| 时间周期 | 活动 | 说明 |
|---------|------|------|
| 每日 | 洞府修炼 + 炼丹/炼器 | 主要经验来源和副职时间 |
| 每周 | 收徒/授课 | 获取声望 + 徒弟进贡 |
| 每月 | 参加高阶交换会 | 以物易物，结交同阶 |
| 每季 | 探索秘境/险地 | 高风险高回报 |
| 每年 | 门派高层会议/势力战争 | 大事件参与 |
| 不定期 | 追杀/被追杀 | PVP 事件 |

**高阶玩家的经济循环：**
```
灵石收入（产业/税收/供奉）
  → 购买顶级材料
    → 炼制高阶丹药/法宝
      → 自用提升实力 或 出售获利
        → 更多灵石 → 更高级材料
```

## 二、社交与互动玩法

### 2.1 师门系统

| 师门功能 | 内容 |
|---------|------|
| 拜师 | 结丹+玩家可收筑基-玩家为徒；NPC 结丹+可收玩家为徒 |
| 师徒任务 | 师傅发布任务 → 徒弟执行 → 双方获奖励 |
| 师徒加成 | 同门派/同师傅玩家组队经验 +10% |
| 师门传承 | 师傅可传授独门心得 → 徒弟突破概率提升 |
| 出师 | 徒弟结丹后可出师 → 获得师傅赠送的本命法宝一件 |

### 2.2 组队系统

| 队伍类型 | 规模 | 收益分配 | 适用场景 |
|---------|------|---------|---------|
| 猎妖小队 | 3-5 人 | 按出力分配 | 野外刷怪、材料收集 |
| 秘境探险队 | 3-8 人 | 队长分配/roll 点 | 副本/秘境探索 |
| 跑商队 | 2-5 人 | 按本金分配 | 跨区域贸易（人多安全） |
| 势力战团 | 10-50 人 | 战功兑换 | 门派战/阵营战 |

### 2.3 论道与交流

| 活动 | 频率 | 收益 |
|------|------|------|
| 门派讲经堂 | 每日 | 少量经验 + 随机感悟 |
| 同阶论道会 | 每周 | 瓶颈突破概率临时提升 |
| 高阶修士讲座 | 每月 | 大量经验 + 特殊功法线索 |
| 藏经阁研读 | 随时 | 花费贡献/灵石阅读功法心得 |
| 切磋比试 | 随时 | 实战经验（无死亡风险） |

### 2.4 洞府与家园

| 洞府等级 | 价格 | 功能 |
|---------|------|------|
| 静室（租） | 10 灵石/月 | 基础打坐修炼 |
| 普通洞府（租/买） | 200 灵石/月 或 5000 灵石买断 | 聚灵阵 + 药圃 + 密室 |
| 灵气洞府 | 2000 灵石/月 | 2x 修炼速度 + 高级药圃 + 炼器室 |
| 灵脉洞府 | 不可购买，需贡献获得 | 3x 修炼速度 + 完整设施 + 灵兽栏 |
| 洞天福地 | 机缘获得 | 独立空间，5x 修炼速度，可种植万年灵药 |

**洞府互动玩法：**
- 可以邀请好友/同道参观洞府
- 洞府可被入侵（高阶玩家可破禁闯入）
- 护府阵法 + 傀儡守卫 = PVE 防御
- 药圃产出灵药 → 炼丹 → 成品出售 → 升级洞府

## 三、副职与生活技能

### 3.1 副职体系总览

| 副职 | 入门要求 | 核心产出 | 后期价值 |
|------|---------|---------|---------|
| 炼丹师 | 炼气期 + 基础丹炉 | 丹药 | 高阶丹药暴利 |
| 炼器师 | 筑基期 + 基础器炉 | 法器/法宝 | 高端装备供不应求 |
| 制符师 | 炼气期 + 符笔符纸 | 符箓 | 符宝是越级战斗的核心 |
| 阵法师 | 筑基期 + 阵法基础 | 阵盘/禁制 | 洞府防御/秘境封锁 |
| 灵植夫 | 炼气期 | 灵药 | 万年灵药的唯一来源 |
| 驭兽师 | 筑基期 | 灵兽/傀儡 | 战斗辅助/材料产出 |
| 矿工 | 无需修为 | 矿石材料 | 装备制造的基础 |

### 3.2 炼丹详解

| 炼丹等级 | 可炼制 | 收益参考 | 升级方式 |
|---------|-------|---------|---------|
| 学徒 | 辟谷丹、小还丹 | 10 灵石/日 | 大量炼制积累经验 |
| 初级 | 养气丹、筑基丹（低概率） | 50 灵石/日 | 成功次数达阈值 |
| 中级 | 凝液丹、高级疗伤丹 | 200 灵石/日 | 成功率 + 丹方数量 |
| 高级 | 结金丹、元婴丹 | 1000+ 灵石/日 | 需要达到结丹境界 |
| 宗师 | 化神丹、大还丹 | 万灵石/日 | 元婴+ 超大量炼制 |

### 3.3 制符详解

| 符箓等级 | 代表产物 | 售价 | 战斗用途 |
|---------|---------|------|---------|
| 初级符箓 | 火弹符、水盾符、神行符 | 1-5 灵石 | 低阶战斗常备消耗品 |
| 中级符箓 | 冰冻符、金刚符、雷爆符 | 10-50 灵石 | 结丹期以下主要输出手段 |
| 高级符箓 | 土牢符、剑影符、大五行符 | 50-500 灵石 | 群体控制/范围伤害 |
| 符宝 | 封印了法宝一击的符 | 500-5000 灵石 | 越级战斗的消耗性王牌 |

**制符的独特机制：**
- 制符速度比炼丹快（可批量生产）
- 符箓是消耗品，市场需求持续且稳定
- 高端符箓（符宝）是筑基/结丹玩家挑战元婴 NPC 的核心手段

## 四、非战斗玩法

### 4.1 探索与发现

| 探索类型 | 难度 | 典型发现 |
|---------|------|---------|
| 古迹探索 | 低-中 | 前人洞府、遗刻功法、残破法器 |
| 险地探查 | 中-高 | 稀有灵药、珍稀材料、秘境入口 |
| 秘境探宝 | 高 | 古宝、通天灵宝、失传丹方 |
| 位面穿梭 | 极高 | 异界材料、跨界机缘 |

**探索触发机制：**
- 场景中随机刷新的可互动物件（发光的石壁、异常灵气波动、神秘的洞穴入口）
- 通过 NPC 传闻/线索触发（坊市闲聊、酒馆密谈中获得提示）
- 修为突破时感知到隐藏区域（如结丹后感知到某地有灵气聚集）

### 4.2 情报与信息

| 信息来源 | 获取方式 | 价值 |
|---------|---------|------|
| 坊市流言 | 与 NPC 闲聊 | 秘境开启时间、商队动向 |
| 酒馆密谈 | 消耗灵石请酒 | 黑市交易情报、仇人行踪 |
| 任务卷轴 | 任务堂/告示栏 | 赏金任务、追杀令 |
| 千里传音 | 高境界传讯术 | 全服告警、大型事件通知 |
| 拍卖预览 | 拍卖行 | 即将拍卖的珍稀物品情报 |

### 4.3 交易与拾荒

| 拾荒类型 | 区域 | 收益 | 风险 |
|---------|------|------|------|
| 废弃矿洞 | 各区域 | 1-10 灵石/天 | 低（偶尔遇妖兽） |
| 战场遗址 | 古战场 | 破碎法器、残存灵石 | 中（可能有怨灵） |
| 妖兽巢穴 | 野外 | 妖兽幼崽、遗骸材料 | 高（母兽护崽） |
| 上古废墟 | 秘境 | 古宝、功法残篇 | 极高（禁制机关） |

## 七、PVP 玩法

### 7.1 切磋（安全 PVP）

| 规则 | 说明 |
|------|------|
| 发起 | 双方同意后在比武台切磋 |
| 死亡 | 不会真死，到 0 血自动认输 |
| 损失 | 无损失，少量修复费 |
| 奖励 | 胜者得少量实战经验 |
| 观战 | 其他玩家可围观下注 |

### 7.2 生死斗（高风险 PVP）

| 规则 | 说明 |
|------|------|
| 发起 | 野外遭遇、仇杀、夺宝 |
| 死亡 | 真死——掉落身上部分灵石 + 随机一件装备 |
| 惩罚 | 凶手声望下降（同门/同道关系恶化） |
| 保护 | 主城内禁止生死斗（禁制保护） |
| 悬赏 | 被杀玩家可发布悬赏令追杀凶手 |

### 7.3 势力战（阵营 PVP）

| 战争类型 | 参战方 | 规模 |
|---------|-------|------|
| 门派资源争夺 | 越国七派间 | 数十人 |
| 正魔大战 | 正道盟 vs 魔道六宗 | 上百人 |
| 星宫之战 | 星宫 vs 逆星盟 | 全区事件 |
| 界面战争 | 灵界各族 vs 魔界 | 全服大型事件（后期） |
