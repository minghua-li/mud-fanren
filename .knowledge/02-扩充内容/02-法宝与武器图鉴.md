# 02：法宝与武器图鉴（游戏设计扩充）

> **本文档是对 [[items/1E-法宝丹药经济]] 中法宝部分的游戏化扩充**，聚焦玩家如何获取、炼制、进阶法宝——法宝分类、各境界限制、获取途径、炼制进阶、相生相克、法宝成长系统。
> 原著法宝数据见 [[items/1E-法宝丹药经济]]，战斗系统见 [[combat/COMBAT_SYSTEM]]。

基于《凡人修仙传》剧本设定，面向 MUD 游戏化设计。本文档的内容属于「合理扩充」范畴。

## 一、法宝总览与分类体系

### 1.1 六大法宝类别

| 类别 | 核心功能 | 装备位置 | 典型代表 |
|------|---------|:-------:|---------|
| ⚔️ **攻击类** | 战斗主输出手段 | 手持/御器 | 飞剑、飞刀、金针 |
| 🛡️ **防御类** | 减伤/护盾/反弹 | 护身 | 古宝盾、灵甲、护心镜 |
| 🌀 **辅助类** | 增益/控制/领域 | 佩戴 | 定风珠、捆仙绳、迷魂幡 |
| 🦅 **飞行类** | 赶路/逃遁/追击 | 踩踏/背负 | 飞舟、灵翼、遁光梭 |
| 🏠 **洞府类** | 随身空间/种植/修炼 | 识海绑定 | 洞天珠、灵兽环 |
| 💎 **特殊类** | 跨界传送/保命/探测 | 特殊 | 破界符、替劫珠 |

### 1.2 法宝品质体系

| 品质 | 名称 | 基础属性倍率 | 可成长性 | 获取难度 |
|:----:|------|:----------:|:--------:|:--------:|
| 1 | 法器 | ×1.0 | 不可成长 | 低 |
| 2 | 灵器 | ×1.5 | 可升1次 | 中低 |
| 3 | 法宝 | ×2.0 | 可升2次 | 中 |
| 4 | 古宝 | ×3.0 | 可升3次 | 高 |
| 5 | 通天灵宝 | ×5.0 | 可升5次 | 极高 |
| 6 | 玄天至宝 | ×8.0 | 完全成长 | 传说 |

### 1.3 法宝属性维度

每件法宝有6项基础属性：

| 属性 | 说明 | 受什么影响 |
|------|------|-----------|
| **攻击力** | 法宝的基础伤害 | 品质×等级×材质 |
| **灵力消耗** | 每次使用消耗的灵力值 | 品质越高消耗越大 |
| **御使境界** | 最低使用境界门槛 | 高品质法宝需求高境界 |
| **炼化时间** | 从获得→完全掌控所需时间 | 与品质正相关 |
| **耐久度** | 使用次数上限，归零需修复 | 日常战斗中损耗 |
| **灵性** | 产生器灵的概率/成长潜力 | 品质越高灵性越高 |

## 二、各境界法宝限制

### 2.1 境界与法宝品级对照

| 玩家境界 | 可用最高品质 | 可御使数量 | 推荐法宝配置 |
|:--------:|:----------:|:---------:|-------------|
| 炼气期 | 灵器（2品） | 1件 | 1攻击法器 |
| 筑基期 | 法宝（3品） | 2件 | 1攻击+1防御 |
| 金丹期 | 古宝（4品） | 3件 | 1攻击+1防御+1辅助 |
| 元婴期 | 通天灵宝（5品） | 4件 | 攻击+防御+辅助+飞行 |
| 化神期 | 通天灵宝（5品） | 5件 | 全类别各1 |
| 炼虚期 | 玄天至宝（6品） | 6件 | 全类别+1特殊 |
| 合体期 | 玄天至宝（6品） | 7件 | 多核心攻击配置 |
| 大乘期 | 玄天至宝（6品） | 8件 | 完全体配置 |

### 2.2 境界突破对法宝的影响

| 突破事件 | 对法宝的影响 |
|---------|-------------|
| 筑基→金丹 | 法器可进阶为灵器（需炼器师协助） |
| 金丹→元婴 | 本命法宝可觉醒器灵 |
| 元婴→化神 | 法宝可跨界面吸收灵气，自动恢复耐久 |
| 化神→炼虚 | 可同时驱动两件本命法宝 |
| 大乘渡劫 | 法宝需承受天劫淬炼（失败则碎） |

## 四、法宝炼制与进阶

### 4.1 炼制流程

```
收集材料 → 学习器谱 → 准备炼器炉 → 熔炼材料 → 塑形 → 
铭刻阵法 → 淬火 → 开光 → 认主
```

### 4.2 炼器材料体系

| 材料类型 | 功能 | 获取难度 |
|---------|------|:-------:|
| **主材**（矿石/灵木/兽骨） | 决定法宝基础属性方向 | 中 |
| **辅材**（精金/秘银/星辰沙） | 提升品质、附加属性 | 高 |
| **魂材**（妖兽魂魄/器灵碎片） | 赋予法宝灵性 | 极高 |
| **阵材**（阵盘/符文石） | 铭刻神通阵法 | 中 |

### 4.3 各品质法宝炼制条件

| 目标品质 | 炼器术要求 | 所需主材品阶 | 成功率 |
|:-------:|:---------:|:----------:|:-----:|
| 法器 | ≥10级 | 1-2阶 | 75% |
| 灵器 | ≥30级 | 3-4阶 | 55% |
| 法宝 | ≥50级 | 5-6阶 | 35% |
| 古宝 | ≥70级 | 7-8阶 | 15% |
| 通天灵宝 | ≥90级 | 9阶+ | 5% |

### 4.4 法宝进阶系统

法宝可通过进阶提升品质，每次进阶保留部分原有属性：

| 进阶路径 | 所需材料 | 成功率 | 属性保留率 |
|---------|---------|:-----:|:---------:|
| 法器→灵器 | 3阶主材×3 + 炼器大师协助 | 60% | 80% |
| 灵器→法宝 | 5阶主材×3 + 本命真火淬炼 | 40% | 70% |
| 法宝→古宝 | 7阶主材×3 + 器灵 | 20% | 60% |
| 古宝→通天灵宝 | 9阶主材×3 + 天劫淬炼 | 10% | 50% |

### 4.5 器灵系统

法宝达到古宝以上品质即可孕育器灵：

| 器灵阶段 | 效果 | 孕育条件 |
|:-------:|------|---------|
| **初生** | 基础属性+10%，自动修复耐久 | 杀敌1000/灵力灌注100小时 |
| **觉醒** | 基础属性+25%，解锁一个法宝技能 | 随主人突破大境界 |
| **化形** | 可短暂化为人形协助战斗 | 灵性值≥90，天材地宝辅助 |
| **圆满** | 法宝通灵，自主护主 | 千年温养+主人合体期 |

### 4.6 炼器术升级

| 等级区间 | 升级方式 | 每次升级需炼制次数 |
|:--------:|---------|:----------------:|
| 1-20 | 炼制法器 | 5次 |
| 21-40 | 炼制灵器 | 10次 |
| 41-60 | 炼制法宝 | 15次 |
| 61-80 | 炼制古宝 | 25次 |
| 81-100 | 炼制通天灵宝 | 50次 |

### 6.2 游戏化改编说明

原著法宝在游戏中遵循以下设计原则：

| 原则 | 说明 |
|------|------|
| **境界解锁** | 核心法宝随玩家境界分阶段解锁能力，非一步到位 |
| **掉落不可得** | 核心法宝不可通过掉落获得，需通过专属任务线 |
| **成长绑定** | 核心法宝与玩家灵魂绑定，不可交易/掉落 |
| **唯一性** | 全服最多一人持有一件核心法宝（掌天瓶全服唯一） |

## 七、法宝养护与消耗

### 7.1 耐久度系统

| 行为 | 耐久消耗 | 说明 |
|------|:-------:|------|
| 普通攻击 | -1 | 每次使用法宝攻击 |
| 承受攻击 | -2 | 用防御法宝格挡 |
| 释放技能 | -3 | 法宝技能消耗更多 |
| 被克制攻击 | -5 | 被五行克制属性攻击 |

### 7.2 修复方式

| 方式 | 修复效果 | 成本 |
|------|---------|------|
| 灵力温养 | 每小时+5耐久 | 少量灵力 |
| 材料修复 | 立即+50耐久 | 同阶材料×1 |
| 炼器师修复 | 立即满耐久 | 灵石（随品质递增） |
| 器灵自愈 | 每分钟+1耐久 | 需要器灵觉醒 |

### 7.3 法宝继承

玩家突破大境界时，可进行法宝继承——保留核心属性，提升品质上限：

| 继承等级 | 消耗 | 属性保留率 |
|:-------:|------|:---------:|
| 炼气→筑基 | 灵石×1000 | 90% |
| 筑基→金丹 | 灵石×5000+材料 | 80% |
| 金丹→元婴 | 灵石×20000+稀有材料 | 70% |
| 元婴→化神 | 灵石×100000+天材地宝 | 65% |

## 八、法宝与各系统联动

### 8.1 与经济系统联动

- **法宝交易市场**：高阶法宝是游戏中价值最高的可交易品
- **材料经济**：法宝炼制/修复拉动矿石/灵木/兽骨等材料需求
- **炼器服务经济**：高阶炼器师为其他玩家提供代炼/修复服务

### 8.2 与战斗系统联动

- **法宝切换CD**：战斗中切换法宝有1-3秒公共CD
- **法宝技能树**：每件法宝有专属技能，通过使用提升熟练度
- **法宝克制**：五行相性在PVP中是关键策略维度
- **法宝损毁**：耐久归零后法宝暂时不可用，需修复

### 8.3 与任务系统联动

- **法宝任务线**：特定法宝有专属获取任务（如掌天瓶任务链）
- **炼器大赛**：定期活动，比拼炼器技艺
- **材料悬赏**：NPC发布特定材料收集任务
- **法宝试炼**：秘境中测试法宝性能，解锁隐藏属性

### 8.4 与社交系统联动

- **法宝展示**：玩家可展示自己的核心法宝（信息面板）
- **炼器师徒**：高阶炼器师收徒传授器谱
- **势力法宝库**：势力共有法宝，成员可借用
- **法宝拍卖会**：定期全服拍卖会，稀有法宝竞拍
