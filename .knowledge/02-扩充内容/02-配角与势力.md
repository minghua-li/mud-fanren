# 02：配角视角与势力深度（合理扩充）

> **本文档是对 [[factions/CHARACTERS]] 和 [[factions/1D-门派种族声望]] 的合理扩充**，聚焦剧本中配角视角下的故事线——当韩立不在场时，那些配角在做什么？他们的动机、日常、支线故事，以及势力之间的日常互动。
> 主力角色清单见 [[factions/CHARACTERS]]，门派结构见 [[factions/1D-门派种族声望]]。

基于《凡人修仙传》设定，面向 MUD 游戏化的配角系统与深度势力设计。

## 一、重要配角的独立故事线

### 1.1 李化元（黄枫谷结丹长老）

**角色定位：** 韩立在黄枫谷的引路人，筑基期导师，结丹中期修士。

#### 背景补充
李化元出身于越国一个没落的修真家族，少年时以真灵根资质被黄枫谷收入门下。苦修百余年才达结丹期，在结丹中期停滞了数十年。他收了韩立为徒，一方面是看中韩立的潜力，另一方面也是为自己的寿元将尽做准备——希望在弟子中找到可托付之人。

#### 支线剧情线索

| 阶段 | 事件 | 玩家可参与方式 |
|------|------|--------------|
| 正魔之战前 | 李化元频繁出入掩月宗（暗恋南宫婉师傅？） | 护送任务、代送信件 |
| 大战期间 | 李化元率黄枫谷弟子防守特定区域 | 加入李化元麾下作战 |
| 战后 | 李化元伤重隐居，托付韩立照看黄枫谷 | 寻药任务、探访故人 |
| 韩立结丹后 | 师徒地位逆转，李化元感慨万千 | 可触发对话事件 |

**支线任务链示例：**
1. 「导师的忧虑」——李化元发现自己的老对头出关，请你打探消息
2. 「故人之后」——李化元故友的后人遇险，请你援手
3. 「最后的心愿」——李化元寿元将尽，请你寻找延寿灵药
4. 「传承」——李化元将毕生心得和高阶丹方托付给你

### 1.2 南宫婉（掩月宗天才女修）

**角色定位：** 掩月宗最出色的女弟子，天灵根，韩立之妻。她是人界少有的以女性身份在修仙界取得顶尖成就的角色。

#### 背景补充
南宫婉出身掩月宗内部的一个修士家族，自幼展现惊人天赋（天灵根）。她外表冷若冰霜，内心却极为重情义。与韩立的关系经历了从合作→暗生情愫→被迫分离→灵界重逢的曲折过程。

#### 南宫婉的独立时间线

| 事件 | 与韩立关系 | 非韩立视角下的情形 |
|------|-----------|-----------------|
| 血色禁地 | 初识，互相戒备 | 南宫婉受宗门指派进入禁地取宝，遇到韩立是意外 |
| 虚天殿 | 再次相遇，联手 | 掩月宗派她争夺虚天鼎，这是宗门最高指令 |
| 正魔大战 | 分离，各自为战 | 南宫婉作为掩月宗高层，指挥门派防御 |
| 飞升灵界 | 重逢 | 她在灵界也经历了自己的历练才走到合体期 |

**玩家可触发的内容：**
- 掩月宗女弟子的专属任务链（南宫婉的师妹/传人路线）
- 听闻关于南宫婉的传闻，触发追踪她的支线
- 在灵界特定区域偶遇南宫婉的故人

### 1.3 元瑶（魔族功法修习者）

**角色定位：** 外表是绝色女子，实为修仙界最罕见的双属性天才。她与韩立彼此欠过人情，关系微妙。

#### 背景补充
元瑶出身不明，幼年被一魔道散修捡到并传授魔道功法。她身具水和暗双属性变异灵根，修炼速度奇快但也备受争议。她行事亦正亦邪，一切以自己变强为优先目标。

#### 支线可能性

| 阶段 | 事件 | 关联维度 |
|------|------|---------|
| 人界 | 元瑶在乱星海和天南之间游走，修炼魔功 | 可提供魔道功法学习机会 |
| 灵界 | 元瑶加入魔族阵营或成为双面间谍 | 玩家在正魔之间的选择分支 |
| 合体期 | 元瑶与韩立在灵界重逢，已是大乘之下有数高手 | 高阶玩家的盟友选择 |

**玩家交互方式：**
- 魔道玩家可触发元瑶的「授业」支线（学习魔族功法）
- 正道玩家面对元瑶时面临「追捕」还是「合作」的选择
- 元瑶不定期出现在各大坊市，出售稀有物品

### 1.4 大衍神君（神识功法创始人）

**角色定位：** 上古时代的顶级修士，大衍诀和炼神术的创造者。在黄枫谷地下密室中以残魂形态存在。

#### 背景补充
大衍神君在万年前曾是灵界最顶尖的存在之一，因炼神术修炼出了岔子，肉身崩溃后元婴逃回人界，躲在黄枫谷的地下密室中苟延残喘。他性格孤僻刻薄，但对得起他的人会得到珍贵的功法传承。

#### 大衍神君的遗产

| 物品/功法 | 获取条件 | 价值 |
|----------|---------|------|
| 大衍诀全本 | 完成大衍神君考验任务链 | 顶级神识功法，可分身控傀儡 |
| 炼神术残篇 | 获得大衍神君信任 | 后期神识碾压同级 |
| 上古情报 | 闲聊触发 | 秘境坐标、古修洞府位置 |
| 傀儡炼制心得 | 完成相关支线 | 傀儡术精通前置 |

**触发方式：**
- 炼气期在黄枫谷后山发现异常灵气波动 → 进入密室 → 初遇大衍神君
- 需要筑基期以上神识强度才能承受大衍诀的第一层传承
- 大衍神君最终会选择「消散」或「夺舍」——取决于你的选择分支

### 1.5 银月/玲珑（妖族公主）

**角色定位：** 敖啸老祖的孙女，七星月体，妖族最出色的后辈。她以银月狼形态跟随韩立多年，后在灵界恢复人身。

#### 独立剧情线索

| 阶段 | 内容 |
|------|------|
| 人界 | 银月作为器灵/灵兽跟随韩立，实际是敖啸老祖安排的守护 |
| 灵界 | 恢复自由身后，银月回到妖族领地，成为七妖王之一的有力候选 |
| 大乘期 | 银月以妖族领袖身份参与灵界战争 |

**玩家可参与的妖族支线：**
- 妖族角色专属任务链（加入银月阵营）
- 人族玩家通过特殊声望进入妖族领地
- 银月发布的猎杀叛徒/清除敌对种族任务

## 二、势力间的日常互动

### 2.1 越国七派的表面和平

越国七派名义上和睦相处，维持了数百年的平衡。但实际上：

| 门派 | 真实态度 | 暗地动作 |
|------|---------|---------|
| 掩月宗 | 自恃最强，对其他六派居高临下 | 暗中渗透清虚门和化刀坞 |
| 黄枫谷 | 中立老好人，其实底牌不少 | 拥有古传送阵秘密，结交乱星海势力 |
| 灵兽山 | 表面中立，实为魔道卧底 | 数百年布局，等待时机接应魔道入侵 |
| 其他四派 | 实力较弱，抱团取暖 | 暗中共建防御联盟防掩月宗一家独大 |

**玩家的体验：**
- 长期驻留某一门派会逐渐察觉到隐藏的门派关系
- 门派关系影响跨门派组队/交易/联姻的难易
- 触发特定事件后玩家可能被迫选择阵营

### 2.2 乱星海的权力平衡

| 势力 | 控制范围 | 经济来源 | 对外策略 |
|------|---------|---------|---------|
| 星宫 | 天星城 + 内星海航线 | 航道税 + 坊市管理费 | 维持现状，压制一切挑战者 |
| 逆星盟 | 外星海部分岛屿 | 劫掠 + 黑市 | 对抗星宫，争取控制权 |
| 妖兽 | 碧灵岛周边海域 | 无（不参与灵石经济） | 防御为主，偶尔袭击人类船只 |
| 散修 | 无固定领地 | 猎妖 + 跑商 | 墙头草，谁给好处跟谁 |

**势力互动事件（玩家可参与）：**
- 星宫发布清剿海盗任务（针对逆星盟附庸）
- 逆星盟发布破坏星宫航线的任务
- 妖兽潮期间双方暂时休战，共抗妖兽
- 拍卖会上可能出现改变势力平衡的宝物（如通天灵宝）

### 2.3 大晋的正魔对峙

大晋的正道盟与魔道六宗之间是真正的你死我活：

| 维度 | 正道盟 | 魔道六宗 |
|------|-------|---------|
| 控制区 | 大晋东部 + 南部 | 大晋西部 + 北部 |
| 核心城市 | 天都（正道盟总部） | 魔渊城（六宗联席议事） |
| 盟友 | 越国七派（名义上） | 天罗国附属宗派 |
| 标志功法 | 正道五行诀、浩然正气 | 鬼灵诀、合欢大法 |
| 对待散修 | 吸纳为外围成员 | 强迫站队，不从则杀 |

**玩家的选择：**
- 加入正道盟 → 与魔道六宗敌对 → 可学正道顶级功法
- 加入魔道六宗 → 与正道盟敌对 → 可学魔道速成功法
- 保持中立 → 两面受压，但可做双面间谍任务

### 2.4 灵界诸族的关系格局

| 关系类型 | 种族对 | 表现形式 |
|---------|-------|---------|
| 同盟 | 人族 ↔ 妖族 | 共抗魔族，天渊城联合驻军 |
| 附庸 | 人族 → 木族 | 木族接受人族保护 |
| 敌对 | 人族 ↔ 角蚩族 | 边境冲突不断，大规模战争偶发 |
| 竞争 | 人族 ↔ 海王族 | 海域资源争夺 |
| 中立 | 人族 ↔ 夜叉/飞灵等 | 互不侵犯，偶有贸易 |

**势力日常事件：**
- 角蚩族劫掠队入侵人族边境 → 悬赏击杀任务
- 木族遭遇不明攻击 → 调查事件真相
- 天渊城轮防 → 玩家可报名参加灵界前线守卫
- 种族间拍卖会 → 稀有跨族物品交易

## 三、非主角视角的重大事件

### 3.1 灵兽山的百年卧底

灵兽山在越国七派中的卧底身份是贯穿人界篇的一条暗线。从灵兽山掌门到普通弟子，大多数成员并不知道自己的宗门是魔道六宗之一御灵宗的棋子。

**卧底的层次：**

| 层级 | 知情程度 | 人数 |
|------|---------|------|
| 掌门 + 太上长老 | 完全知情，与御灵宗直接联系 | 2-3 人 |
| 核心长老 | 知道宗门与魔道有关联，但不知全部计划 | 5-8 人 |
| 普通长老 | 完全不知情，被蒙在鼓里 | 20+ 人 |
| 弟子 | 完全不知情 | 数千人 |

**事件触发点：**
- 玩家在灵兽山修炼到筑基后期时，可能发现宗门异常
- 通过特定支线可以揭发灵兽山的卧底身份
- 选择「揭发」→ 正魔大战提前爆发，越国格局剧变
- 选择「加入」→ 倒向魔道阵营，开启魔道玩家路线

### 3.2 乱星海拍卖会与暗流

乱星海的百年一次的大型拍卖会不仅仅是交易场所，更是各方势力角力的舞台。

| 参与者 | 目的 | 行动 |
|-------|------|------|
| 星宫 | 展示实力，拉拢高阶散修 | 拿出珍稀拍品，安排托儿竞价 |
| 逆星盟 | 刺探情报，争取盟友 | 派卧底混入，暗中接触大买家 |
| 散修 | 出售秘境所得，购买突破丹药 | 真实参与拍卖 |
| 妖兽（化形） | 收购人族法器 | 伪装成散修入场 |

**拍卖会上可能发生的事件：**
- 买家被盯上 → 出城后被劫杀（PVP 事件）
- 拍品真假争议 → 需要鉴定技能
- 神秘买家（可能是高阶 NPC）不计代价竞拍某物 → 暗示该物品有隐藏价值

### 3.3 魔界渗透的暗线

魔界对灵界的入侵并非突如其来的战争，而是经历了漫长的渗透阶段：

| 阶段 | 时间跨度 | 表现形式 |
|------|---------|---------|
| 第一阶段 | 数千年前 | 空间裂缝偶然出现，少量低阶魔族渗透 |
| 第二阶段 | 千年前 | 魔界势力开始在灵界建立据点 |
| 第三阶段 | 故事主线时间 | 大规模入侵开始，天渊城建成 |
| 第四阶段 | 后期 | 全面战争，各族被迫联手 |

**玩家可参与的渗透反制：**
- 发现异常灵气波动 → 追踪魔族间谍
- 捣毁魔族在灵界的秘密据点
- 收集魔界情报 → 上交天渊城换取功勋

## 四、剧情选择分支系统

### 4.1 玩家的阵营归属

| 选择轴 | 选项 | 影响 |
|-------|------|------|
| 正魔归属 | 正道 / 魔道 / 中立 | 可学功法、可结交 NPC、区域准入 |
| 门派归属 | 越国七派之一 | 师门任务、同门关系、门派战争 |
| 韩立关系 | 友善 / 中立 / 敌对 | 后期可得机缘或成为阻碍 |
| 势力选择 | 星宫 / 逆星盟 / 保持独立 | 乱星海阶段的阵营任务 |

### 4.2 关键剧情选择节点

| 节点 | 选择 | 后果 |
|------|------|------|
| 血色禁地 | 帮南宫婉 / 独吞宝物 | 影响后续与掩月宗的关系 |
| 正魔之战 | 支援正道 / 加入魔道 / 逃避 | 改变整个中期的阵营地图 |
| 虚天殿夺宝 | 参与争夺 / 放弃 / 做渔翁 | 是否获得补天丹的关键 |
| 灵兽山秘密 | 揭发 / 沉默 / 加入 | 改变越国势力格局 |
| 大衍神君结局 | 帮助消散 / 助其夺舍 / 夺取传承 | 获得功法或与强者为敌 |

### 4.3 声望的多维系统

每个势力有独立声望值，影响可交互内容：

| 声望等级 | 所需声望值 | 解锁内容 |
|---------|----------|---------|
| 冷漠（初始） | 0 | 基础对话 |
| 友善 | 100 | 购买普通物品 |
| 信任 | 500 | 接日常任务 |
| 尊敬 | 2000 | 购买稀有物品 + 特殊任务 |
| 崇敬 | 5000 | 学功法 + 购买顶级物品 |
| 崇拜 | 10000 | 特殊称号 + 唯一道具 + 阵营特权 |

## 五、势力产业的日常运转

### 5.1 灵石矿脉的管理

| 管理方式 | 控制者 | 产出分配 |
|---------|-------|---------|
| 门派直营 | 大门派 | 全部上缴门派，统一分配 |
| 家族承包 | 修真家族 | 家族自留 70%，上缴门派 30% |
| 散修自由开采 | 无主矿脉 | 谁采到归谁，但需防抢劫 |
| 税收分成 | 大势力控制区 | 私人开采者上缴 20% 给控制势力 |

### 5.2 坊市的运营

| 坊市类型 | 运营方 | 收入来源 |
|---------|-------|---------|
| 固定坊市（如太南谷） | 当地最强势力 | 摊位费 + 交易税 |
| 流动坊市 | 商会组织 | 中介费 |
| 黑市 | 散修联盟或魔道 | 不收税但无保障 |
| 线上拍卖 | 广源斋（跨位面） | 手续费，以物易物 |

**坊市 NPC 管理：**
- 每个坊市有一个「坊主」NPC（金丹或元婴期）
- 有专门的「巡查」NPC 维护秩序（禁止在坊市内动武）
- 坊市公告板发布近期的求购/出售信息

### 5.3 修仙家族的生存策略

修仙家族是介于散修和门派之间的中间形态：

| 策略 | 适用家族 | 特点 |
|------|---------|------|
| 依附大门派 | 中小家族 | 定期上贡 → 获得保护 |
| 联姻扩张 | 势力相当者 | 通过婚姻合并两族资源 |
| 独立发展 | 有元婴老祖的大家族 | 不依附任何门派，但风险高 |
| 商业贸易 | 擅长经商的家族 | 放弃武力争夺，专注经济 |

**玩家与家族的互动：**
- 可加入修仙家族成为客卿
- 完成家族任务提升在家族中的地位
- 娶/嫁入家族 → 获得家族专属功法/资源
- 高阶玩家可建立自己的修真家族

## 六、NPC 日常 AI 行为

### 6.1 NPC 的日常行程

| NPC 类型 | 典型日程 |
|---------|---------|
| 门派普通弟子 | 早课 → 执事堂接任务 → 执行任务 → 回门派交任务 → 打坐 |
| 门派长老 | 修炼 → 偶尔授课 → 处理门派事务 → 炼丹/炼器 |
| 坊市商人 | 开店 → 补货 → 接待顾客 → 闭店结算 |
| 散修 | 自由安排，可能在野外猎妖/采集/探宝 |
| 赏金猎人 | 接悬赏 → 追杀目标 → 领赏 → 消费 |

### 6.2 NPC 互动事件

| 互动类型 | 触发条件 | 内容 |
|---------|---------|------|
| 求助 | 玩家实力与事件难度匹配 | NPC 请玩家帮忙送信/猎妖/寻人 |
| 交易 | 玩家持有 NPC 需要的物品 | 触发隐藏交易选项 |
| 挑战 | 玩家声望过高/过低 | NPC 主动挑战或挑衅 |
| 情报 | 特定区域/时间 | NPC 透露秘境/宝藏线索 |
| 恩怨 | 玩家之前帮过/得罪过该 NPC | 后续回报/报复 |

### 6.3 随机 NPC 生态事件

| 事件 | 概率 | 表现 |
|------|------|------|
| 妖兽袭击商队 | 低 | 野外偶遇，可救援获益 |
| 修士斗法 | 中 | 两个 NPC 在野外厮杀，可选择围观/介入/渔翁得利 |
| 坊市纠纷 | 低 | NPC 在坊市中争吵，可调解或煽风点火 |
| 秘境入口发现 | 极低 | NPC 发现秘境入口，消息扩散引发争夺 |
| 仇杀 | 随机 | 高阶 NPC 追杀低阶 NPC，体现修仙界的残酷 |

## 七、深度势力玩法

### 7.1 门派贡献与晋升路线

| 身份 | 要求 | 特权 |
|------|------|------|
| 外门弟子 | 入门 | 基础设施使用权 |
| 内门弟子 | 筑基 + 贡献 500 | 中级功法 + 洞府 |
| 真传弟子 | 筑基后期 + 贡献 2000 + 长老推荐 | 核心功法 + 个人洞府 |
| 执事 | 筑基 + 贡献 3000 | 管理权限 + 俸禄翻倍 |
| 长老 | 结丹 + 贡献 10000 | 山头管理 + 收徒权 |
| 太上长老 | 元婴 + 贡献 50000 | 宗门决策权 + 最高功法 |

### 7.2 跨势力外交

| 外交行为 | 消耗 | 效果 |
|---------|------|------|
| 遣使通好 | 灵石 1000 | 两势力关系 +10，持续 30 天 |
| 联姻 | 双方同意 | 关系 +50，持续 1 年 |
| 共同讨伐 | 双方出兵 | 关系 +30，击败目标后 +50 |
| 经济制裁 | 封锁贸易 | 目标势力收入 -20%，关系 -30 |
| 宣战 | 不消耗 | 关系清零，进入战争状态 |

### 7.3 领地建设与争夺

| 领地等级 | 功能 | 争夺方式 |
|---------|------|---------|
| 矿脉 | 灵石产出 | 限时 PVP 占据 |
| 灵药园 | 灵药产出 | 限时采集权 PVP |
| 灵脉 | 修炼加速 | 长期占据（需持续防守） |
| 城市 | 税收 + 坊市 | 势力战攻城 |
