# 02：丹药体系详解（游戏设计扩充）

> **本文档是对 [[items/1E-法宝丹药经济]] 中丹药部分的游戏化扩充**，聚焦玩家如何获取、炼制和使用丹药——丹药分类、配方材料、品级效果、炼制门槛、药性冲突、各境界丹药配置。
> 原著丹药数据见 [[items/1E-法宝丹药经济]]，经济循环见 [[02-经济与资源]]。

基于《凡人修仙传》剧本设定，面向 MUD 游戏化设计。本文档的内容属于「合理扩充」范畴——剧本提及但未展开的游戏机制细节。

## 一、丹药总览与分类体系

### 1.1 六大丹药类别

丹药按功效分为六大类，每类在玩家成长路径中扮演不同角色：

| 类别 | 核心用途 | 典型代表 | 玩家关注点 |
|------|---------|---------|-----------|
| 🧪 **修为丹** | 增加修为值/突破成功率 | 筑基丹、结金丹、化神丹 | 修炼加速核心消耗品 |
| ❤️ **疗伤丹** | 恢复气血/疗伤/解毒 | 回春丹、续命丹、清毒散 | 战斗续航必需品 |
| ⚡ **临时强化** | 短时间提升属性 | 狂暴丹、护体丹、疾风丹 | 高难战斗/副本策略品 |
| 🔄 **永久增益** | 永久提升某项属性 | 洗髓丹、易经丹、开窍丹 | 长线养成的战略投资 |
| 🧠 **辅助丹** | 突破瓶颈/感悟功法 | 破障丹、悟道丹、通灵丹 | 卡境界/卡功法时的钥匙 |
| 💀 **毒丹** | 战斗用毒/暗算 | 蚀骨散、迷魂烟、化功散 | PVP/任务特殊场景 |

### 1.2 品级体系

每颗丹药有五个品级维度，决定最终效果：

| 品级 | 名称 | 效果系数 | 服用境界门槛 | 炼制难度 |
|:----:|------|:--------:|:-----------:|:--------:|
| 1 | 凡品 | ×1.0 | 无 | 入门 |
| 2 | 良品 | ×1.5 | 筑基 | 初学 |
| 3 | 上品 | ×2.0 | 金丹 | 熟练 |
| 4 | 极品 | ×3.0 | 元婴 | 大师 |
| 5 | 仙品 | ×5.0 | 化神+ | 传说 |

丹药品质在炼制时随机产出，受炼丹术等级、材料品质、丹方熟练度影响。

## 二、配方与材料体系

### 2.1 丹方构成

每个丹方包含三要素：主药 + 辅药 + 药引

| 成分 | 作用 | 示例 |
|------|------|------|
| **主药**（1-2味） | 决定丹药大类与核心功效 | 千年灵芝（修为类主药） |
| **辅药**（2-4味） | 调节品级、中和药性、增加成功率 | 百年人参、雪山莲 |
| **药引**（1味） | 决定丹药具体品种与特殊效果 | 妖兽精血（种类决定效果方向） |

### 2.2 材料来源

| 来源 | 产出类型 | 稳定性 | 玩家获取方式 |
|------|---------|:------:|-------------|
| 灵草种植 | 基础草药（1-3阶） | 高 | 洞府药圃种植，需种子+灵田 |
| 野外采集 | 中高阶灵草（2-6阶） | 中 | 探索地图特定区域，有刷新CD |
| 妖兽掉落 | 妖丹/精血/特殊材料 | 低 | 击杀妖兽后概率掉落 |
| 秘境探索 | 珍稀材料（5-9阶） | 极低 | 限时秘境/副本产出 |
| 坊市购买 | 全阶材料（溢价） | 高 | NPC商店/玩家摆摊 |
| 门派贡献 | 门派特有材料 | 中 | 完成师门任务兑换 |

### 2.3 材料品阶与境界对应

| 材料阶位 | 适合炼制的丹药品级 | 材料刷新区域 | 采集所需境界 |
|:--------:|:-----------------:|-------------|:-----------:|
| 1阶 | 凡品 | 新手村周边 | 炼气期 |
| 2阶 | 凡品-良品 | 人界普通区域 | 炼气后期 |
| 3阶 | 良品 | 人界危险区域 | 筑基期 |
| 4阶 | 良品-上品 | 人界秘境/乱星海 | 筑基后期 |
| 5阶 | 上品 | 灵界普通区域 | 金丹期 |
| 6阶 | 上品-极品 | 灵界危险区域 | 元婴期 |
| 7阶 | 极品 | 灵界秘境/禁区 | 化神期 |
| 8阶 | 极品-仙品 | 仙界区域 | 炼虚期 |
| 9阶 | 仙品 | 仙界禁地/上古遗迹 | 合体期+ |

## 四、炼制系统

### 4.1 炼制流程

```
收集材料 → 学习/拥有丹方 → 准备丹炉 → 生火控温 → 投放主药 → 投放辅药 → 
投放药引 → 凝丹阶段（QTE） → 开炉取丹
```

### 4.2 丹炉系统

| 丹炉类型 | 品级上限 | 成功率加成 | 获取方式 |
|---------|:-------:|:---------:|---------|
| 黄铜炉 | 凡品 | +0% | 新手赠送 |
| 玄铁炉 | 良品 | +5% | 坊市购买（500灵石） |
| 地火炉 | 上品 | +10% | 门派贡献兑换 |
| 天火鼎 | 极品 | +15% | 秘境掉落/拍卖 |
| 上古神炉 | 仙品 | +25% | 专属任务线 |

### 4.3 成功率的计算公式

```
最终成功率 = 基础成功率 ×（1 + 丹炉加成 + 炼丹术等级×0.02）
             × 材料品质系数 - 丹药品级难度罚值
```

| 变量 | 说明 |
|------|------|
| 基础成功率 | 丹方固定值（凡品70%，良品50%，上品30%，极品15%，仙品5%） |
| 丹炉加成 | 见上表 |
| 炼丹术等级 | 每级+2% |
| 材料品质系数 | 全部用对应品级材料=×1.0，混入低品级材料=×0.5~0.8 |
| 品级难度罚值 | 每提升一品级 -10% |

### 4.4 炼丹术升级

| 等级区间 | 升级方式 | 每次升级需炼制次数 |
|:--------:|---------|:----------------:|
| 1-20 | 炼制凡品丹药 | 5次 |
| 21-40 | 炼制良品丹药 | 8次 |
| 41-60 | 炼制上品丹药 | 12次 |
| 61-80 | 炼制极品丹药 | 20次 |
| 81-100 | 炼制仙品丹药 | 50次 |

### 4.5 凝丹环节（QTE玩法）

炼制最后阶段进入凝丹QTE，玩家需在30秒内完成：

1. **控火**：灵力条波动，点击保持指针在绿色区间（±15%）
2. **注灵**：连续输入灵力（频率越快，品质越高）
3. **收丹**：在正确时机（丹药成型闪光的瞬间）点击收丹

QTE完成度影响最终品质：
- 完美（3/3）：品质+1级概率+30%
- 良好（2/3）：品质不变
- 失败（≤1/3）：品质-1级，失败率+20%

## 五、药性冲突与丹毒系统

### 5.1 丹毒累积机制

每服用一颗丹药都会累积丹毒值：

| 丹药品级 | 单颗丹毒值 | 自然衰减速率 |
|:--------:|:---------:|:-----------:|
| 凡品 | 5点 | 每小时-2点 |
| 良品 | 10点 | 每小时-3点 |
| 上品 | 20点 | 每小时-4点 |
| 极品 | 40点 | 每小时-5点 |
| 仙品 | 80点 | 每小时-6点 |

### 5.2 丹毒层级与惩罚

| 丹毒值区间 | 状态名称 | 惩罚效果 |
|:---------:|---------|---------|
| 0-50 | 正常 | 无 |
| 51-150 | 微毒 | 修炼效率-10%，气血恢复-10% |
| 151-300 | 中毒 | 修炼效率-25%，气血恢复-25%，概率中毒 |
| 301-500 | 重毒 | 修炼效率-50%，气血恢复-50%，持续掉血 |
| 500+ | 毒入膏肓 | 每日损失修为，无法修炼，需稀有解毒丹 |

### 5.3 化解丹毒方式

| 方式 | 效果 | 限制 |
|------|------|------|
| 自然排毒 | 随时间缓慢衰减 | 极慢，高阶丹药需要数天 |
| 运功逼毒 | 每小时-50点 | 消耗大量灵力，无法战斗 |
| 解毒丹 | 立即-200点 | 有冷却时间（12小时） |
| 特殊功法 | 排毒效率+100% | 需修炼特定功法 |
| 天材地宝 | 立即清零 | 极其稀有 |

### 5.4 丹药同效冲突规则

| 规则 | 说明 |
|------|------|
| **同类别叠加惩罚** | 30分钟内服用同类别丹药，第二颗效果-50%，第三颗-75% |
| **永久增益唯一性** | 一生限服次数内有效，超出无效 |
| **强化冲突** | 同时服用2种以上临时强化丹，效果取最高值，其他无效 |
| **药性相克** | 寒性丹药与热性丹药同时服用，双方效果-50% |
| **酒药冲突** | 饮酒后30分钟内服用丹药，效果-30%，中毒概率+50% |

## 七、丹药与各系统联动

### 7.1 与经济系统联动

- **炼丹产业链**：材料采集→炼制→成品销售，形成完整产业链
- **价格弹性**：突破丹（筑基丹/结金丹）在版本初期价格最高，随供应增加逐步下降
- **代炼服务**：高阶炼丹师可为低阶玩家代炼，收取手续费（市场调节）
- **材料期货**：稀有材料（如万年灵乳）存在囤货/炒作空间

### 7.2 与战斗系统联动

- **战斗中用药**：回春丹/狂暴丹等可在战斗中使用（占用一回合行动）
- **丹药CD**：各类丹药有独立冷却时间，战斗中无法连续吞服
- **药性爆发**：特定丹药组合触发额外效果（如狂暴丹+五行丹=五行狂暴）
- **丹药反制**：战斗中使用解毒丹可反制对手的毒攻

### 7.3 与任务系统联动

- **门派丹方任务**：完成门派任务获得专属丹方
- **药材采集任务**：定时刷新采集任务，产出特定材料
- **炼丹大赛**：定期活动，比拼炼丹技艺
- **丹方残卷收集**：上古丹方拆分为残卷，集齐可合成完整丹方

### 7.4 与社交系统联动

- **师徒传丹**：师傅可赠与徒弟部分丹药（降低新手门槛）
- **丹道交流会**：玩家组队讨论丹方/交易材料
- **炼丹师徒**：高阶玩家可以收炼丹学徒（共享部分熟练度收益）
- **势力丹房**：势力共有丹炉，成员可低价使用

## 八、炼丹师成长路线

### 8.1 专属身份线

1. **炼丹学徒**（炼丹术1-20）：炼制凡品，成功率低，自给自足
2. **炼丹师**（炼丹术21-40）：可炼良品，开始接代炼订单
