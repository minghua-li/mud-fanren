# 1D：门派/种族/声望系统设计

> 基于《凡人修仙传》剧本提取，面向 LPC MUD 实现的势力体系设计文档。

## 一、概述

凡人修仙传的世界横跨人界、灵界、魔界三界，势力体系极为庞大。本设计文档定义门派组织结构、种族特性分类以及声望机制的运作规则，为 LPC 代码实现提供完整的数值框架与数据结构设计。

```
三界势力总览：
人界 ←—————————— 灵界 ←—————————— 魔界
  │                      │                  │
  越国七派(炼气~元婴)   灵界各族(化神~大乘)  真魔始祖
  乱星海势力             人族/妖族/角蚩等    圣祖/魔尊
  大晋正道盟/魔道六宗    天渊城/圣皇殿       普通魔族
  凡人江湖               广源斋              魔界入侵主线
```

## 二、人界门派体系

### 2.1 越国七大门派

越国七大门派是支撑整个越国修仙界的参天大树，七派共同把持越国修仙界资源。

| 门派 | 实力排名 | 属性倾向 | 核心特点 | 代表人物 | 声望初始关系 |
|------|---------|---------|---------|---------|------------|
| **掩月宗** | 最强 | 正道/中立 | 双修之术，女修为主，天月神舟 | 南宫婉（元婴期） | 中立 |
| **灵兽山** | 第二 | 中立偏邪 | 驱兽役虫，魔道卧底 | 云露（结丹期） | 中立→敌对(正魔战后) |
| **黄枫谷** | 中等 | 正道 | 主角师门，太岳山脉，传送阵秘密 | 令狐老祖、李化元 | 友善(韩立师门) |
| **清虚门** | 中等 | 正道 | 道家传承，清心寡欲 | — | 中立 |
| **化刀坞** | 中等 | 中立 | 炼器工艺突出 | — | 中立 |
| **天阙堡** | 中等 | 中立 | 建筑/阵法擅长，黄色巨印服饰 | — | 中立 |
| **巨剑门** | 中等 | 正道 | 全男弟子，黑衣巨剑 | — | 中立 |

**门派实力梯度设计：**
- 掩月宗是七派中唯一有两位元婴修士的门派
- 灵兽山有秘密背景（实为魔道六宗布下的棋子）
- 其余五派实力相当，各自坐拥一州资源

#### 2.1.1 各派详细设定

**黄枫谷（韩立师门）**
- 位置：建州西部，太岳山脉中部
- 规模：弟子一万余人（炼气期>90%、筑基期数百人、结丹期寥寥数人、元婴期一人）
- 内部结构：
  - 议事大殿（三层大门，数十米高石殿）
  - 传功阁（授功法，分内外两层）
  - 百药园（韩立看守的药园）
  - 岳麓殿（法器、丹药配方、密术收藏）
  - 迎宾楼 / 坊市（太岳山脉东北边缘）
- 关键人物：令狐老祖（元婴）、李化元（结丹，韩立师父）、红拂（结丹）
- LPC 门派标记：`"sect": "huangfeng_valley"`

**掩月宗**
- 越国第一大派，女弟子容貌上等，提倡双修之术
- 标志法器：天月神舟（巨型飞行法器）
- 灵根测试体系最为严谨
- 关键人物：南宫婉（元婴期，女主角）
- LPC 门派标记：`"sect": "yanyue_sect"`

**灵兽山**
- 驱兽役虫闻名，弟子服饰花哨，皮囊口袋众多
- 暗中为魔道六宗势力，正魔大战中倒戈
- 驯养多种灵兽/灵虫（金翅蚕、噬金虫等）
- LPC 门派标记：`"sect": "lingshou_mountain"`

**清虚门 / 化刀坞 / 天阙堡 / 巨剑门**
- 清虚门：道家传承，清心寡欲，术法为主
- 化刀坞：刀法传承，炼器工艺突出
- 天阙堡：建筑与阵法擅长
- 巨剑门：剑修传承，全为男弟子

### 2.2 乱星海势力

乱星海是海外修仙世界，独立于天南大陆的法力体系。

| 势力 | 统领者 | 实力范围 | 声望初始关系 | 说明 |
|------|--------|---------|------------|------|
| **星宫** | 天星双圣（元婴后期） | 内星海 | 中立 | 统治乱星海数千年，有星宫禁地 |
| **逆星盟** | 万天明、六道极圣 | 内星海 | 中立 | 反星宫联盟，实力与星宫相当 |
| **碧灵岛妖兽** | 蛟龙一族 | 外星海 | 中立/敌对 | 1-10级妖兽系统 |
| **散修联盟** | 散修自发组织 | 各岛 | 友善 | — |
| **天星城** | 星宫直属 | 内星海中心 | 中立 | 乱星海最大修仙城市 |

**乱星海关键设计：**
- 星宫控制着通往外星海的传送阵
- 逆星盟由不满星宫统治的散修组成
- 外星海是妖兽地盘，也是试炼/探险的主要区域
- 天星城有大量商铺、拍卖行

### 2.3 大晋势力

大晋是人界第一大国，面积约为天南地区的十倍。

| 势力 | 属性 | 实力 | 说明 |
|------|------|------|------|
| **正道盟** | 正道 | 七大派联盟 | 大晋正道修仙者联盟，风都国为大本营 |
| **魔道六宗** | 魔道 | 六大宗门 | 天罗国为大本营，与正道盟对峙 |
| **大晋皇室** | 中立 | 世俗+修仙 | 世俗帝国统治者内部也有修仙力量 |

**正魔对峙格局：**
- 正道盟七派 vs 魔道六宗，实力基本相当
- 越国地处正魔交界地带，成为双方争夺的缓冲区域
- 魔道六宗暗中在越国布局（灵兽山就是棋子）
- 正魔大战爆发 → 越国七派被迫选边

**魔道六宗详情：**
| 宗名 | 擅长 | 特征 |
|------|------|------|
| 合欢宗 | 双修魅惑 | 弟子多俊男美女，擅长阴阳采补 |
| 天煞宗 | 杀戮功法 | 以杀证道，煞气越重功法越强 |
| 鬼灵门 | 鬼道/炼尸 | 操控鬼物，炼制尸傀 |
| 御灵宗 | 御兽 | 与灵兽山有关联，驭使灵兽战斗 |
| 天魔宗 | 天魔功法 | 召唤天魔附体，短时暴增战力 |
| 阴罗宗 | 毒功/暗杀 | 用毒、暗杀、情报 |

### 2.4 凡人江湖势力

| 势力 | 类型 | 描述 |
|------|------|------|

### 2.5 九国盟与慕兰草原

- 九国盟：天南最南部九国联盟，抵御慕兰法士入侵
- 慕兰法士：修炼体系不同于修仙者，以法术为主
- 法士战争：慕兰草原势力与九国盟的长期消耗战

## 三、门派等级体系（LPC 通用结构）

每个门派内部应有统一的弟子等级结构：

| 等级 | 称号 | 权限 | 对应修为 | 说明 |
|------|------|------|---------|------|
| 0 | 外门弟子 | 基础功法、基础任务 | 炼气1-6层 | 入门阶段 |
| 1 | 内门弟子 | 核心功法、师承 | 炼气7-13层 | 通过门派考核 |
| 2 | 真传弟子 | 镇派绝学、秘地出入 | 筑基期 | 门派核心培养对象 |
| 3 | 执事/管事 | 管理权、资源分配 | 筑基后期~结丹 | 负责具体事务 |
| 4 | 长老 | 决策权、收徒权 | 结丹期 | 管理层 |
| 5 | 太上长老 | 最高决策 | 元婴期 | 门派真正支柱 |
| 6 | 掌门/宗主 | 一票否决 | 元婴期 | 名义最高权力者 |

**加入与退出门派规则：**
- 加入条件：修为≥炼气3层+门派贡献≥0+无叛门记录
- 内门晋升：修为≥炼气7层+完成进阶任务
- 退出门派：声望-5000，贡献清空
- 背叛门派：声望-20000，全门派追杀

**LPC 数据结构：**

## 五、魔界势力

### 5.1 魔界层级体系

| 层级 | 境界对应 | 权力范围 | 代表角色 |
|------|---------|---------|---------|
| **真魔始祖** | 真仙级别 | 魔界最高存在 | — |
| **圣祖** | 大乘期 | 统领一方魔域 | 元魇始祖、宝花、血光圣祖 |
| **魔尊** | 合体期 | 魔界中层领主 | — |
| **魔王** | 炼虚期 | 区域管理者 | — |
| **高阶魔族** | 化神期 | 基层将领 | — |
| **普通魔族** | 元婴期以下 | 基层战力 | — |
| **低阶魔物** | 结丹期以下 | 炮灰 | 魔猿、魔狼等 |

### 5.2 魔界入侵主线设计

**阶段一：裂缝初现**
- 灵界各处出现小规模空间裂缝
- 低阶魔族通过裂缝涌入
- 各势力开始警觉

**阶段二：局部冲突**
- 魔界先锋部队入侵灵界边缘区域
- 人族与妖族联军抵抗
- 天渊城成为前线要塞

**阶段三：全面战争**
- 圣祖级魔族降临灵界
- 各族被迫联合（木族附庸人族、角蚩族暂时休战）
- 大规模界战爆发

**阶段四：最终决战**
- 韩立等顶级修士反攻魔界
- 真魔始祖苏醒
- 决定两界命运

## 六、声望系统

### 6.1 声望类型总览

凡人修仙传 MUD 采用多层次声望体系：

| 声望类型 | 范围 | 影响 | 存储位置 |
|---------|------|------|---------|
| **正道声望** | 全局 | 正道门派好感基础值 | 玩家对象 |
| **魔道声望** | 全局 | 魔道/邪道门派好感基础值 | 玩家对象 |
| **门派声望** | 具体门派 | 该门派的任务/商店/功法 | 玩家对象(mapping) |
| **种族声望** | 具体种族 | 该种族的互动权限 | 玩家对象(mapping) |
| **区域声望** | 具体区域 | 该区域的进入/活动权限 | 玩家对象(mapping) |
| **势力声望** | 具体组织 | 该组织的专属内容 | 玩家对象(mapping) |

### 6.2 声望等级与数值阈值

| 等级 | 名称 | 数值范围 | 效果 |
|------|------|---------|------|
| -6 | 死敌 (Mortal Enemy) | -100000以下 | 见面即攻击，无法交涉 |
| -5 | 血仇 (Blood Feud) | -100000 ~ -50000 | 触发追杀令，全地图通缉 |
| -4 | 敌对 (Hostile) | -50000 ~ -10000 | NPC拒绝对话，可主动攻击 |

```lpc
// 声望增益计算函数原型
// 返回值: 折扣比例(0.0~1.0)，1.0=原价
float query_discount(string faction_id, int reputation) {
    if (reputation >= 100000) return 0.50;  // 传说 - 5折
    if (reputation >= 50000)  return 0.60;  // 崇拜 - 6折
    if (reputation >= 20000)  return 0.70;  // 崇敬 - 7折
    if (reputation >= 5000)   return 0.80;  // 尊敬 - 8折
    if (reputation >= 1000)   return 0.90;  // 信任 - 9折
    if (reputation >= 100)    return 0.95;  // 友善 - 9.5折
    if (reputation > -100)    return 1.00;  // 中立 - 原价
    if (reputation > -1000)   return 1.50;  // 冷淡 - 1.5倍
    if (reputation > -5000)   return 2.00;  // 不友好 - 2倍
    return 3.00;                             // 憎恶以下 - 3倍
}
```

```lpc
int check_diplomacy_change(string faction_a, string faction_b) {
    int relation = faction_relations[faction_a][faction_b];
    string status;
    
    if (relation >= 80) status = "同盟";
    else if (relation >= 40) status = "友好";
    else if (relation >= 10) status = "友善";
    else if (relation >= -10) status = "中立";
    else if (relation >= -40) status = "紧张";
    else if (relation >= -70) status = "敌对";
    else status = "死敌";
    
    return status;
}
```

## 八、势力战争系统

### 8.1 门派战

**触发条件：**
- 两派关系降至"敌对"以下
- 资源争夺点被对方占据超过7天
- 门派声望任务链的最终事件

**战斗模式：**
- 战区：指定地图区域（如矿脉/灵脉）
- 参战：双方在线弟子均可参与
- 胜负：累计击杀积分/占领据点
- 奖励：胜方获得资源点控制权+全体声望+500

**LPC 数据结构：**
```lpc
struct clan_war_data {
    string aggressor;        // 进攻方门派ID
    string defender;         // 防守方门派ID
    string battlefield;      // 战区地图
    int start_time;          // 开战时间戳
    int duration;            // 持续时间(秒)
    mapping score;           // ([门派ID: 积分])
    string status;           // "preparing" / "active" / "ended"
};
```

### 8.2 正魔之战

正魔之战是覆盖整个天南地区的大规模势力冲突。

**触发流程：**
1. 魔道六宗向越国渗透（灵兽山倒戈为信号）
2. 正道盟号召各派组成联军
3. 越国七派被迫选择阵营
4. 大规模战争爆发

**参与奖励：**
| 贡献 | 奖励 |
|------|------|
| 击杀10名敌对弟子 | 正道/魔道声望+1000 |
| 摧毁敌方据点 | 对应势力声望+5000 |
| 击杀敌方结丹修士 | 声望+10000 |
| 击杀敌方元婴修士 | 声望+50000+特殊称号 |
