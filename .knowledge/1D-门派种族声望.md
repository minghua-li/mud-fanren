# 1D：门派 / 种族 / 声望系统设计

基于剧本《凡人修仙传》（忘语著）全文提炼。本文档聚焦三界（人界→灵界→魔界）的门派体系、种族设定、跨势力声望/阵营关系。

---

## 第一部分：人界门派体系

### 1.1 越国七大门派

越国修仙界由七大门派把持，属于中立偏正阵营。

| 门派 | 实力 | 属性倾向 | 特点 | 代表性人物 |
|------|------|---------|------|-----------|
| **掩月宗** | 最强 | 中立偏正 | 提倡双修之术，女弟子多为容貌上等；标志性法器天月神舟 | 霓裳仙子、南宫婉 |
| **黄枫谷** | 中等 | 正道 | 韩立所在门派，山门在建州太岳山脉中部；弟子万余人 | 令狐老祖(元婴)、钟灵道(掌门)、李化元(结丹)、红拂(结丹) |
| **灵兽山** | 第二 | 中立偏邪 | 擅长驱兽役虫，实际为魔道六宗卧底，正魔大战中倒戈 | — |
| **清虚门** | 中等 | 正道 | 道门传承，清心寡欲；飞行法器为雪虹绫 | 无游子 |
| **化刀坞** | 中等 | 中立 | — | — |
| **天阙堡** | 中等 | 中立 | 黄色大印法宝 | — |
| **巨剑门** | 中等 | 正道 | 全部男子，黑衣，背负一人高无鞘巨剑 | — |

**越国七派的历史背景：**
- 千年前越国正邪对立，七派当时只是不入流的小门派，靠做墙头草生存
- 正邪双方一场惨烈大战后实力大损，七派趁机联手将正邪道统连根拔起
- 如今七派的功法混杂正邪两派，属于中立阵营
- 一旦有外部正邪势力入侵，七派立即联手击退

### 1.2 魔道六宗（天罗国）

天罗国魔道六宗是天南地区两大超级势力之一，实力远超越国七派。

| 宗门 | 特点 | 说明 |
|------|------|------|
| **鬼灵门** | 驱鬼役妖、毒术、暗术 | 六宗中最弱，但实力仍远超掩月宗；少门主为暗灵根 |
| **合欢宗** | 双修功法 | 正魔大战中活跃 |
| **御灵宗** | 御兽之术 | 与越国灵兽山有隐秘联系，数百年布局 |
| 其余三宗 | 未详述 | 剧本中未详细展开名称 |

**魔道六宗实力描述：**
- 天罗国面积是越国三倍以上
- 六宗中的鬼灵门(最弱)实力也远超掩月宗(越国最强)
- 六宗入侵时，通过内部策反(姜国、车骑国各宗派被收买)快速攻占两国
- 灵兽山实为御灵宗卧底，布局数百年

### 1.3 正道盟（风都国）

| 势力 | 说明 |
|------|------|
| **风都国正道盟** | 天南另一大超级势力，与魔道六宗对峙 |
| **实力** | 与魔道六宗旗鼓相当，同为天南两大势力 |
| **行为** | 趁魔道入侵越国时同时扩张，入侵相邻中等国家 |

### 1.4 周边国家修仙势力

| 国家 | 修仙势力 | 说明 |
|------|---------|------|
| **元武国** | 天星宗、神兵门、万妙观 | 正魔均势；天星宗以阵法闻名，神兵门以炼器闻名，万妙观以符道闻名 |
| **紫金国** | 多家门派 | 越国七派的援军来源国 |
| **姜国** | 多家小宗派 | 最先被魔道六宗吞并；部分宗派被收买，部分被灭 |
| **车骑国** | 多家小宗派 | 与姜国一同沦陷；金鼓原正魔大战主战场在此交界 |

### 1.5 九国盟

- **构成：** 九个国家组成的紧密联盟
- **位置：** 天南大陆南部
- **目的：** 共同抵御慕兰族法士入侵
- **核心城市：** 虞国·阗天城（天南唯一的修士之城）
- **历史沿革：** 原越国七派败退后被吸收；韩立师父李化元战死于法士争斗
- **后期加入：** 越国七派残余势力、其他被驱逐宗派和散修
- **四大势力格局：** 正魔大战后，天南形成正道盟/魔道六宗/九国盟/天道盟四足鼎立

### 1.6 天道盟

- 由正魔大战中剩余的中小国家组成
- 与正道盟、魔道六宗、九国盟形成三足鼎立（后期）

## 第二部分：修仙家族体系

### 2.1 越国修仙家族

越国修仙家族借助七大门派才可在修仙界生存，是盘缠在七派大树上的枝蔓。

| 家族 | 关联门派 | 说明 |
|------|---------|------|
| **燕家** | 原出鬼灵门 | 越国第一大修仙家族，有结丹期修士坐镇；后归顺鬼灵门 |
| **万家** | 巨剑门 | 祖先是散修后入巨剑门；家规不歧视散修 |
| **叶家** | 黄枫谷（叶师叔） | 黄枫谷内部掌权家族之一 |
| **陈家** | 七派 | 陈巧倩所在家族；陈家先祖为黄枫谷弟子 |
| **墨府** | 凡人江湖 | 墨大夫家族，惊蛟会控制者；非修仙家族 |

### 2.2 元武国修仙家族

| 家族 | 说明 |
|------|------|
| **付家** | 元武国最大修仙家族之一，与韩立结仇 |
| **齐家** | 普通家族，齐云霄曾被逐出 |

### 2.3 修仙家族的地位特点

- 修仙家族是七派的枝蔓，需依附门派生存
- 升仙令是各派发给有功修仙家族的特殊令牌
- 家族子弟可通过升仙大会或升仙令入门派
- 家族中也有"天灵根"等天才出现
- 正魔大战后，越国修仙家族分三种态度：归顺魔道、随六派撤离、观望

### 3.2 妖兽进阶体系

妖兽按等级划分，1-10级对应修士境界：

| 妖兽等级 | 对应修士境界 | 说明 |
|---------|-------------|------|
| 1-3级 | 炼气/筑基 | 低阶妖兽，如推山兽(1级下阶)、寒冰蟾(1级下阶)、火焰鼠(1级下阶) |
| 4-6级 | 结丹 | 中阶妖兽，妖丹价值极高 |
| 7级 | 元婴初期(化形初阶) | 化形雷劫后部分化为人形 |
| 8级 | 元婴中期(化形中阶) | 更高化形程度，如毒蛟 |
| 9级 | 元婴后期/化神(化形高阶) | 高阶化形，如裂风兽 |
| 10级+ | 炼虚+ | 超阶妖兽 |

**妖兽分类：**
- **猛兽类**：虎、狼、豹等
- **飞禽类**：雷鹏、裂风兽等
- **毒虫类**：蜈蚣、毒蝎等（同阶中最难缠）
- **鳞甲类**：蛟龙、龟等（防御力最强）
- **灵禽类**：速度优势

**蛟龙的特殊性：**
- 天地灵兽，从上古时期留存
- 修行速度远超普通妖兽
- 神通远超同阶，可越阶灭敌
- 八级蛟龙≈九级普通妖兽

## 第四部分：魔界势力

| 名称 | 层级 | 说明 |
|------|------|------|
| **元魇始祖** | 圣祖 | 魔界顶级存在 |

### 4.3 魔界相关设定

- 魔界通过空间裂缝入侵灵界是小说后期主线
- 魔族功法另成体系（魔气 vs 灵气）
- 部分魔族可与修仙者达成契约
- 古魔修设下奇阵禁制，撕裂空间裂缝
- 空间裂缝中阴冥之地可吸入不同海域的修士

## 第六部分：声望系统设计

### 6.1 势力声望等级

| 等级 | 名称 | 说明 |
|------|------|------|

// 玩家对象上的声望记录（存储在 F_DBASE 中）
// query("reputation/<faction_id>") 返回声望值 -10000~10000

// 声望等级判定函数
int query_reputation_level(string faction) {
    int rep = query("reputation/" + faction);
    if (rep <= -5000) return -3;  // 死敌
    if (rep <= -2000) return -2;  // 敌对
    if (rep <= -500)  return -1;  // 不友好
    if (rep < 500)    return 0;   // 中立
    if (rep < 2000)   return 1;   // 友好
    if (rep < 5000)   return 2;   // 信任
    if (rep < 8000)   return 3;   // 尊敬
    return 4;                      // 崇拜
}

## 第九部分：设计要点总结

### 9.1 核心设计原则

1. **三界渐进**：从人界江湖→修仙界→灵界→魔界，势力复杂度逐级递进
2. **阵营明确**：正魔对立＋中立灰色地带，阵营非绝对
3. **种族多样**：灵界30+种族各有特色，避免同质化
4. **声望驱动**：每个势力独立的声望体系，声望影响任务/商店/功法/领地访问

### 9.2 实现优先级建议

| 优先级 | 内容 | 说明 |
|--------|------|------|
| **P0** | 越国七派 + 凡人江湖 | 新手期核心内容 |
| **P1** | 魔道六宗 + 元武国势力 | 中期内容扩展 |
| **P2** | 乱星海势力 | 中后期海洋扩展 |
| **P3** | 九国盟 + 慕兰法士 | 后期大陆扩展 |
| **P4** | 大晋帝国势力 | 后期超大地图 |
