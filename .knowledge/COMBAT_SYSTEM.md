# 战斗体系设计（COMBAT_SYSTEM）

> **详细 LPC 实现见** → [[1F-法术剑诀阵法战斗]]

## 一、五行法术体系

### 五行属性

| 属性 | 特点 | 代表性法术 | 克制关系 |
|------|------|-----------|---------|
| **金** | 攻击凌厉、锐利 | 金刃术、金刚罩 | 金→木→土→水→火→金（相克） |
| **木** | 生命力、恢复、缠绕 | 缠绕术、回春术 | 相生：金生水→水生木→木生火→火生土→土生金 |
| **水** | 控制、变化、防御 | 水盾术、冰封术 | — |
| **火** | 爆发、范围伤害 | 火弹术、连珠火球 | — |
| **土** | 防御、沉重 | 土墙术、地刺术 | — |

### 法术等级

| 等级 | 描述 | 使用者 |
|------|------|--------|
| **初级下阶** | 火弹术、冰弹符级别 | 炼气期 |
| **初级中阶** | 连珠雷符、缠绕术 | 炼气后期-筑基 |
| **初级上阶** | 御风诀、清灵散 | 筑基期 |
| **中级法术** | 大范围五行法术 | 结丹期+ |
| **高级法术** | 领域类、禁咒类 | 元婴期+ |
| **法则类** | 时间/空间/生死法则 | 大乘/真仙 |

## 二、剑诀体系

剑修是修仙世界的主流输出职业之一。

### 韩立的剑修进阶路径（参考）

| 阶段 | 境界 | 飞剑数 | 剑诀形态 |
|------|------|--------|---------|
| 初成 | 筑基 | 6口青竹蜂云剑 | 基本飞剑御使 |
| 剑阵入门 | 结丹 | 12→36口 | 大庚剑阵 |
| 剑阵精进 | 元婴 | 72口 | 青蟠剑阵/无形剑气 |
| 灵界进阶 | 炼虚-合体 | 剑阵+元合五极山 | 五极山剑阵 |
| 终极形态 | 大乘 | 玄天斩灵剑 | 法则级剑诀 |

### 游戏中剑修的模块
- **飞剑品质**：法器级→法宝级→灵宝级→通天灵宝→玄天之宝
- **飞剑属性**：金/木/水/火/土 + 变异属性
- **剑阵组合**：多口飞剑可组成不同剑阵，产生质变
- **剑诀心法**：对应功法的剑诀子系统

## 四、辅助系统

### 神识系统
- **作用**：探查隐形、抵挡夺舍、控制多个目标（傀儡/法宝）
- **提升途径**：大衍诀、炼神术
- **战斗应用**：神识冲击可打断施法、探知对方意图

### 阵法系统

| 类型 | 示例 | 用途 |
|------|------|------|
| 防御阵 | 天罡北斗阵 | 洞府/驻地防御 |
| 攻击阵 | 九宫天乾符阵 | 大范围杀伤 |
| 辅助阵 | 聚灵阵 | 加速灵气恢复 |
| 传送阵 | 古传送阵 | 跨区域传送 |
| 困阵 | 颠倒乾坤阵 | 控制敌人 |
| 封印阵 | 太阴玄玉封魔阵 | 魔族封印 |

### 傀儡/灵兽系统

**傀儡等级：**
- 低阶傀儡（木质/石质）→ 筑基
- 中阶傀儡（金属）→ 结丹
- 高阶傀儡（通灵傀儡）→ 元婴-化神
- 真仙级傀儡（蟹道人）→ 顶级

**灵兽系统：**
- 收服妖兽为灵兽（噬金虫、银翅夜鹏、啼魂兽等）
- 灵兽可培养、进阶
- 战斗中可选择让灵兽参战

## 五、战斗特殊机制

### 夺舍机制
- 只有高境界可对低境界施展
- 一生只能一次
- 失败后果：元神反噬
- 游戏化设计：PVP/PVE中的终极终结技，使用后角色永久改变

### 心魔
- 突破大境界时触发
- 心魔强度由业力/行为决定
- 可收集心魔劫相关道具降低难度

### 天劫
- 化神（飞升）、大乘（大天劫）、大乘大圆满（飞升劫）
- 天劫威力由积累决定
- 可提前准备渡劫法宝/阵法/丹药

## 六、PVP与PVE设计

### PvE（怪物战斗）
- 普通怪物 → 基于境界差自动战斗
- 精英怪物 → 需要战术配合
- BOSS级 → 组队副本
- 世界BOSS → 全服事件

### PvP（玩家对战）
- 切磋模式（点到为止）
- 生死斗（可击杀，有掉宝/声望惩罚）
- 门派战（阵营PvP）
- 领地战（势力争夺）

## 七、技能系统结构

每个功法/法术作为独立LPC对象：
```
skill/<属性>/<功法名>.c
special/<功法名>/<绝招名>.c
```

技能属性结构（参考现有LPC风格）：
```c
set("name", "青元剑诀");
set("type", "剑诀");
set("difficulty", 50);          // 修炼难度
set("attr", "金/木");           // 属性
set("level", 60);               // 最高等级
set("skill_class", "剑修");     // 分类
// 绝招列表
set("perform", ([
  "feijian"  : "basic_attack",
  "jianzhen" : "sword_array",
  "jianyu"   : "sword_rain",
]));
```
