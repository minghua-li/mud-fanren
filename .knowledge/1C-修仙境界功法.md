# 1C：修仙境界 / 灵根 / 功法系统设计

> **本文档是 [[CULTIVATION_SYSTEM]] 的详细数值框架版**，包含境界经验曲线、灵根乘数、瓶颈概率模型、寿元系统等 LPC 实现所需数据。
> 体系总览见 [[CULTIVATION_SYSTEM]]。
>
> 基于剧本《凡人修仙传》提炼，面向 LPC MUD 实现的数值框架设计。

## 一、境界体系（Cultivation Realm System）

### 1.1 三大境界划分

原著将修仙境界分为**下境界、中境界、上境界**三层（见剧本第 4285 行）：

| 层级 | 包含境界 | 对应界面 | 说明 |
|------|---------|---------|------|
| **下境界** | 炼气 → 筑基 → 结丹 → 元婴 → 化神 | 人界 | 人界修士的完整修炼路径 |
| **中境界** | 炼虚 → 合体 → 大乘 | 灵界 | 飞升灵界后的修炼路径 |
| **上境界** | 渡劫 → 真仙 | 仙界通道 | 大乘圆满后渡天劫飞升 |

### 1.2 各境界详细设定（基于原著）

#### 炼气期（Qi Refinement）

| 属性 | 数值 |
|------|------|
| 层数 | 1-13 层 |
| 寿命 | ~100 年（常人寿元几乎无增长） |
| 特征 | 基础功法修炼，感应灵气打通经脉 |
| 阶段划分子层 | 1-3 层入门、4-6 层初成、7-9 层中阶、10-13 层高阶 |
| 战力参照 | 第 9 层可施展完整版火弹术约百次（剧本 3135 行） |

**原文依据**（4307-4311 行）：
- 炼气期修士突破到筑基期，需基础功法达到第 7 层以上，并服用筑基丹
- 伪灵根者一般一生止步于炼气期 3-4 层
- 十名炼气期修仙者中只有一人能在筑基丹帮助下进入筑基期
- 长春功在黄枫谷散修坊市的售价为 2 低阶灵石（全本 1-13 层）

#### 筑基期（Foundation Establishment）

| 属性 | 数值 |
|------|------|
| 子境界 | 初期 / 中期 / 后期 |
| 寿命 | ~200 年 |
| 特征 | 可施放先天真火（炼丹炼器的道家罡火），可完全发挥符宝威力 |
| 门派地位 | 黄枫谷约一万弟子中筑基期仅数百人，是中坚力量 |
| 结丹比例 | 一百名筑基期修士也不一定有一个人能结丹成功 |
| 新手法力参照 | 筑基初期的韩立苦练大衍诀数年才达到筑基中期（剧本 10932 行） |

**原文依据**（5528 行）：
- "先天真火是筑基期修士才可具有的道家罡火"
- 筑基后可使用神识凝炼法，将符宝威力全部发挥
- 筑基是真正"踏入修仙界的门槛"（剧本 5218 行）

#### 结丹期（Core Formation / Golden Core）

| 属性 | 数值 |
|------|------|
| 子境界 | 初期 / 中期 / 后期 |
| 寿命 | ~500-600 年 |
| 特征 | 可炼制法宝、可制作符宝、可真正驱物飞行 |
| 法宝 | 青竹蜂云剑（全套 72 口飞剑）、乾蓝冰焰等 |
| 地位 | 黄枫谷仅寥寥数人，决定门派存亡 |
| 结丹方式 | 天灵根者无瓶颈自动结丹；普通修士需雪灵水+天火液等灵物辅助 |

**原文依据**（17579 行）：
- 结丹期寿元达五六百年
- 丹田真火化为三昧真火
- 青元剑诀第 7 层起可用"剑影分光术"神通

#### 元婴期（Nascent Soul）

| 属性 | 数值 |
|------|------|
| 子境界 | 初期 / 中期 / 后期 |
| 寿命 | ~1000+ 年 |
| 特征 | 元婴不灭则不死、可元婴出窍；法术质变 |
| 战力 | 一只脚踏入元婴期的修士可横扫结丹期（如极阴祖师） |
| 突破条件 | 碎丹成婴 + 心魔劫；需补天丹等奇物辅助 |

**原文依据**（29050 行）：
- 韩立以不到 200 岁结婴，属极年轻
- 补天丹可辅助结丹修士凝练元婴（剧本 21919 行）

#### 化神期（Deity Transformation）

| 属性 | 数值 |
|------|------|
| 子境界 | 初期 / 中期 / 后期（大圆满飞升） |
| 寿命 | ~2000+ 年 |
| 特征 | 被界面排斥，必须飞升灵界；人界天花板 |
| 分布 | 整个越国无一人达到化神期（剧本 4287 行） |
| 基本设定 | 化神期修士被迫脱离本界前往灵界（剧本 4296 行） |

#### 中境界三阶（灵界阶段）

| 境界 | 寿命 | 核心特征 |
|------|------|---------|
| **炼虚期** | ~5000+ 年 | 化神后吸纳灵界灵气，合五行之气 |
| **合体期** | ~10000+ 年 | 法体合一，神与身合 |
| **大乘期** | ~数万年 | 积累法力 → 每万年一次大天劫 → 渡飞升劫 |

注：中境界是原著第二部（灵界篇）核心内容，MUD 中可作为后期扩展内容设计。

### 1.3 经验值曲线设计建议

**境界突破成功率模型（基于原文数据推导）**：

| 突破路径 | 成功率 | 原文依据 |
|---------|--------|---------|
| 炼气 → 筑基（服筑基丹） | ~10% | "十个炼气期修仙者中，只有一个可以在筑基丹帮助下进入筑基期" |
| 筑基 → 结丹 | ~1% | "一百个筑基期的人，也不一定能有一人可结丹成功" |
| 天灵根者结丹 | 豁免瓶颈（100%可达） | "天根人的无瓶颈就可结丹" |
| 变异灵根结丹 | 无豁免，但速度接近天灵根 | 剧本 4317 行 |

**经验曲线斜率建议**：
- 炼气期：每层递增 20% 经验需求（1-13 层快速入门）
- 筑基期：每子境界递增 50% 经验需求
- 结丹期：每子境界递增 100% 经验需求
- 元婴及以上：指数级增长

## 二、灵根系统（Spiritual Root System）

### 2.1 灵根分类（基于原著精确分类）

#### 按灵根属性数分

| 类型 | 属性数 | 修炼速度 | 结丹瓶颈 | 出现概率 | 原文支撑 |
|------|--------|---------|---------|---------|---------|
| **伪灵根** | 4-5 种混杂 | 极慢（正常修炼止步炼气 3-4 层） | 极难 | 绝大多数 | 剧本 4307-4308 行："修炼效果惨不忍睹，只能练至三、四层" |
| **真灵根** | 2-3 种属性 | 正常（约伪灵根的 2-3 倍） | 正常瓶颈 | 约总灵根者的 20% | 只有两三种属性，"修炼起来较为快速" |
| **天灵根** | 1 种纯属性 | 2-3 倍普通速度 | **无瓶颈**（自动结丹） | 数百年一遇 | 剧本 4309-4313 行 |
| **变异灵根** | 变异属性 | 接近天灵根 | 正常瓶颈（但战力更强） | 每二三十年出现一个 | 剧本 4314-4318 行 |

#### 天灵根详解

原文 4309-4313 行详尽描述了天灵根的逆天之处：
- 修炼速度是普通灵根的 2-3 倍
- 修炼到筑基期顶峰时不需面对结丹瓶颈，**可轻易开始结丹**
- "每当天根人出现时，往往就被各个修仙门派疯狂争抢"
- "相当于平白为本门预定好了一个结丹期的大高手"
- 出现几率"基本上每数百年才可被修仙门派发现一个"

#### 变异灵根详解

原文 4314-4318 行：

| 变异灵根 | 基础组合 | 特点 |
|---------|---------|------|
| **雷灵根** | 金 + 水变异 | 攻击力最强，克制妖魔 |
| **冰灵根** | 土 + 水变异 | 冰冻控制，防御力强 |
| **风灵根** | 基础属性变异 | 速度极快，身法优异 |
| **暗灵根** | 特殊变异 | 隐匿、吞噬类能力（如鬼灵门少主） |

变异灵根的关键特征（原文 4317 行）：
- 修炼速度不差于天灵根
- 无豁免结丹瓶颈天赋
- 如找到匹配功法，一顶三四个同等实力普通修仙者
- 各修仙门派极为欢迎

#### 特别案例：三阳之体

剧本 9438 行记载了李化元的特殊体质"三阳之体"（仅次于天灵根）：
- 修炼顶阶功法《真阳决》如鱼得水
- 以此资质成功结成金丹
- 可作为游戏中特殊体质设定的参考

### 2.2 灵根检测与生成

**原文中的灵根检测方式**：
- 修仙门派使用专门的测试方法检测灵根属性（剧本 5108-5111 行）
- 韩立在黄枫谷入门时由王师叔亲自测试，"灵根属性四属性缺金，属于伪灵根"
- 测试结果为"资质低下"，修炼至九层已"高的出奇"

**灵根出现的概率体系**（原文 4320 行）：
- 一万个人中才可能有一个生有灵根的人
- 五六个有灵根的人中只能找出一个真灵根的人
- 伪灵根占修仙者中的绝大多数

**血脉遗传机制**（原文 4322-4328 行）：
- "灵根极容易出现在相同血脉继承的人身上"
- 修仙门派安排弟子在世俗界成家立业，后代灵根比率大增
- 久而久之形成了"修真家族"
- 修真家族和修仙大派是修仙界的两大势力来源

### 2.3 灵根对游戏机制的影响

| 维度 | 影响描述 | 数值建议 |
|------|---------|---------|
| 修炼速度 | 基础经验获取倍率 | 伪灵根=0.3x / 真灵根=1.0x / 天灵根=2.5x / 变异灵根=2.5x |
| 瓶颈突破 | 大境界突破概率修正 | 天灵根无瓶颈自动突破；变灵根无额外修正 |
| 功法学习 | 可学功法的属性限制 | 灵根属性决定可学功法系别（金木水火土+变异） |
| 战力加成 | 变异灵根同阶战力加成 | 变异灵根玩家若学匹配功法，战力为同阶普通玩家的 3-4 倍 |
| 结丹概率修正 | 伪灵根结丹概率趋近于 0 | 伪灵根需极致丹药积累+特殊机缘（如三转重元功）才有望结丹 |

### 2.4 后天改变机制（游戏可扩展）

原文提及的后天灵根改善途径：
- 补天丹：可"弥补人五行灵根的东西"（剧本 19505 行），能洗练灵根
- 洗灵池、净灵莲：特殊天材地宝可洗练提升灵根品质
- 天雷竹等剧毒属性灵根：韩立的修为提升得益于小瓶中灵药的积累

### 3.2 功法品阶体系

| 品阶 | 适用境界 | 代表作 | 法术特点 |
|------|---------|-------|---------|
| **基础功法** | 炼气-筑基期 | 长春功、基础五行诀 | 炼气打坐、精进法力、辅助性强 |
| **中等功法** | 筑基-结丹期 | 青元剑诀、大衍诀、真阳决 | 有明确的攻防神通 |

### 3.3 核心功法详解（基于原文）

#### 长春功（Eternal Spring Technique）

| 属性 | 内容 |
|------|------|
| 属性 | 木系基础功法 |
| 品阶 | 基础 |
| 修炼层数 | 1-13 层（完整版） |
| 市价 | 2 低阶灵石（黄枫谷散修坊市） |
| 韩立起始 | 第 4 层（1年）→第 5 层→第 6 层→第 8 层→第 9 层→第 11 层→第 13 层 |
| 核心效果 | 每层提升精神力、头脑、五感（视觉/听觉/嗅觉），但不增强体质（原文 1415 行） |
| 特点 | 无法用于实战和厮杀（原文 1101 行）；但可为后续修仙打下根基 |
| 最终 | 第 13 层大圆满后可尝试筑基 |

**经验曲线**：
- 韩立大量服药从第 4 层到第 8 层，再到第 11 层（每次精进需大量丹药黄龙丹+金髓丸）
- 第 11 层后黄龙丹和金髓丸彻底失效（剧本 5437 行），需更高级的丹方

#### 青元剑诀（Azure Origin Sword Technique）

| 属性 | 内容 |
|------|------|
| 属性 | 木系剑修功法 |
| 品阶 | 中等（完整版全 13 层可达化神） |
| 黄枫谷版本 | 残本 9 层（前 6 层筑基可练，后 3 层需结丹） |
| 完整版 | 13 层全本（韩立从金色光字获得，含三转重元功辅助） |
| 修炼层数 | 9 层（残本）/ 13 层（完整含三层剑影分光） |

**层数与神通对应**（原文 8324-8330 行）：

| 层段 | 层数范围 | 可修炼者 | 解锁神通 |
|------|---------|---------|---------|
| 前三层 | 1-3 | 炼气期 | 剑芒（远程剑气攻击） |
| 中三层 | 4-6 | 筑基期 | 护体剑盾（防御+剑芒反击） |
| 后三层 | 7-9 | 结丹期 | 剑影分光术（每升一层多一道剑影，最多三道） |

**青元剑诀的缺陷**（剧本 8376-8389 行）：
- 第 4 层起每隔数日散功：流失新炼法力的 1/10
- 第 5 层流失 2/10，第 6 层流失 3/10
- 实际修炼者在第 6 层时法力总量反而比其他修士多 1/3（因为打下的根基更厚）
- 黄枫谷只出过一个修炼到第 6 层的弟子

**配套法宝**：青竹蜂云剑
- 以天雷竹为主材料的成套飞剑（最多 72 口为一套）
- 带有辟邪神雷属性，克制妖魔
- 韩立以 72 口飞剑纵横修仙界

**配套辅助**：三转重元功
- 第 6 层散功重修，共三次（三转）
- 每转压缩真元，使液化真元向固化靠拢
- 三转完成后可绕开结丹瓶颈（原文 9521 行："回壁结丹期瓶颈的惊天奇效"）
- 实际韩立在完成二转后即成功结丹（原文 17564 行）

#### 大衍诀（Great Development Art）

| 属性 | 内容 |
|------|------|
| 属性 | 神识修炼功法（无属性） |
| 品阶 | 高阶 |
| 层数 | 4 层（韩立修炼到第 3 层） |
| 所属 | 千竹教开派祖师独创（剧本 9099 行） |
| 核心功能 | 强大神识 + 分神之术 |
| 操控协助 | 操纵傀儡、神识附体 |
| 结丹辅助 | 修炼至高层后结丹比普通修士"轻松得多"（剧本 9112 行） |
| 修炼难度 | 第 2 层花费多年，第 3 层花费二十多年（原文 17571 行） |
| 韩立成就 | 第 3 层神识即可媲美元婴期修士 |

**游戏机制建议**：
- 非战斗主功法，为辅助/傀儡型分支
- 增加神识上限、控制傀儡数量、神识探查范围
- 高阶后可分神同时操控多个目标
- 第 4 层以上神识可媲美真仙级（炼神术后）

#### 明王诀（Wrathful King Art）

| 属性 | 内容 |
|------|------|
| 属性 | 佛门护法功法（偏魔道） |
| 品阶 | 高阶 |
| 来源 | 佛门金刚护法修炼功法（原文 40709 行） |
| 基础 | 由妖族炼体功法改良而来（原文 40731 行） |
| 核心效果 | 化解煞气、法体双修 |
| 配套 | 需金刚舍利子辅助 |
| 进阶 | → 三头六臂魔神法相（与梵圣真魔功融合） |

#### 梵圣真魔功（Brahma Holy True Devil Art）

- 法体双修顶级功法（化神以上阶段使用）
- 源自古魔功法，经佛门妖族改造
- 可现三头六臂魔神法相
- 核心战力型功法

#### 炼神术（Spirit Refining Art）

- 顶级神识功法
- 第三层神识可媲美真仙（剧本 98 行）
- 神识分化、控制傀儡、探查范围极大

|------|------|------|------|

### 3.4 功法体系游戏设计框架

```lpc
// 功法数据结构建议
class technique {
    string name;            // 功法名称
    string attribute;       // 五行属性
    int level;              // 当前修炼层数（1-13/9/4）
    int max_level;          // 最大层数
    int grade;              // 品阶（1=基础 2=中等 3=高阶 4=顶级）
    mapping skills;         // 神通技能列表
    mapping bonuses;        // 被动属性加成
    int mana_multiplier;    // 法力系数
    int breakthrough_rate;  // 突破加成
}
```

**功法养成**：
- 层数突破需要满足：修为门槛 + 灵根要求 + 丹药辅助 + 概率判定
- 同种功法可改修（如从残本转修完整本）
- 变异灵根匹配功法时获得战力加成

## 四、瓶颈机制（Breakthrough System）

### 4.1 瓶颈分类与突破方式

原著中明确的瓶颈突破路径：

| 突破方式 | 适用场景 | 原文支撑 |
|---------|---------|---------|
| **丹药辅助** | 所有瓶颈突破 | 筑基丹、雪灵水+天火液、结金丹等 |
| **机缘突破** | 大境界瓶颈 | 秘境历练、高人灌顶（如血色禁地） |
| **战斗突破** | 小瓶颈 | 生死一线间激发潜力（原文 24123 行） |
| **水到渠成积累** | 天灵根专属 | 天灵根者无瓶颈自动结丹 |
| **特殊功法** | 结丹瓶颈专用 | 三转重元功可绕开结丹瓶颈 |

### 4.2 关键突破丹药体系

#### 筑基丹（Foundation Establishment Pill）
- 原文最核心的筑基突破丹药
- 练气期第 7 层以上 + 筑基丹 = 有几率筑基
- 越国七大派联手每十年才出千余颗
- 升仙大会胜出者 + 升仙令持有者可获得配额
- 一粒筑基丹使伪灵根筑基成功率约 1%（原文 5110 行）
- 对纯粹精进法力也有作用（可提升 2-3 层基础功法）

#### 结丹辅助灵物
- **雪灵水** + **天火液**：结丹时必须服用的两种灵物（剧本 16906 行）
- 三转重元功二转 + 大量丹药 → 结丹成功率大幅提升

#### 其他辅助丹药

| 丹药名称 | 作用阶段 | 功效 |
|---------|---------|------|
| 黄龙丹/金髓丸 | 炼气期（前中期） | 精进法力，突破小瓶颈 |
| 辟谷丹/定神符 | 通用辅助 | 辅助修炼状态 |
| 补天丹 | 结丹-元婴 | 洗练灵根、凝练元婴 |
| 长生丹 | 炼虚以上 | 延寿、突破大瓶颈 |
| 化神丹 | 化神期辅助 | 原著未详述 |

### 4.3 瓶颈概率模型

```lpc
// 瓶颈突破判定公式（建议）
float breakthrough_chance(string from_realm, string to_realm, object player) {
    float base_chance = REALM_TRANSITION[from_realm][to_realm].base;
    float root_bonus = player->spirit_root_bonus();
    float pill_bonus = player->pill_effect("breakthrough");
    float technique_bonus = player->technique_bonus();
    
    // 天灵根结丹=自动成功
    if (player->spirit_root() == "天灵根" && to_realm == "结丹")
        return 1.0;
    
    return base_chance * root_bonus * (1 + pill_bonus + technique_bonus);
}
```

**关键突破概率参考（基于原文推导）**：

| 突破路径 | 基础概率 | 丹药加成 | 灵根修正 |
|---------|---------|---------|---------|
| 炼气 → 筑基 | 10%（有筑基丹） | 多粒服用可叠加 | 天灵根无加成（已自动结丹） |
| 筑基 → 结丹 | 1% | 雪灵水+天火液+大量灵药 | 天灵根=100%；大衍诀高层=增加 |
| 结丹 → 元婴 | 极低 | 需要有补天丹等奇物 | 变灵根无豁免 |
| 元婴 → 化神 | 接近不可能（人界内） | 需要飞升灵界的机缘 | — |

### 4.4 修行境界的心魔设计

元婴期碎丹成婴时必经**心魔劫**（剧本第 12 行）：
- 突破过程中的精神对抗
- 心魔强度与玩家业力/煞气值相关
- 明王诀（佛门功法）可有效化解煞气反噬（剧本 40734 行）

**游戏实现建议**：
- 突破结丹/元婴时触发心魔战斗（意识对抗）
- 玩家煞气值越高，心魔越强
- 修炼对应功法、服用清明类丹药可降低心魔难度

## 五、寿命系统（Lifespan System）

### 5.1 各境界寿命基准值（基于原文）

| 境界 | 寿命 | 原文出处 |
|------|------|---------|
| 凡人/练武 | ~60-80 年 | 世俗界基准 |
| 炼气期 | ~100 年 | 原文 4288-4291 行：修仙者寿命与境界息息相关，每层境界提升都使寿命成倍增长 |
| 筑基期 | ~200 年 | 韩立筑基后担忧寿命悬殊（剧本 10022 行） |
| 结丹期 | ~500-600 年 | 原文 17579 行 |
| 元婴期 | ~1000+ 年 | 原文 29050 行 |
| 化神期 | ~2000+ 年 | — |
| 炼虚期 | ~5000+ 年 | — |
| 合体期 | ~10000+ 年 | — |
| 大乘期 | ~数万年 | 每万年一次大天劫 |

### 5.2 延寿机制

| 方式 | 说明 | 原文依据 |
|------|------|---------|
| **境界突破** | 每提升一个大境界，寿命大幅增加 | 最根本的延寿方式 |
| **寿元果** | 虚天殿特产，可用于炼制长生丹，延寿五六十年 | 剧本 19392-19396 行 |
| **长生丹** | 用寿元果炼制的延寿丹药 | 同上 |
| **补天丹** | 有传言可增寿元（实际以凝练元婴为主） | 剧本 21919 行 |
| **特殊功法** | 玄魂炼妖大法可突破普通修士的寿命限制 | 剧本 19171 行 |
| **万年灵乳** | 微量延寿 | — |

**关键规则**：突破大境界是增加寿命上限的唯一可靠路径，其他丹药延寿最多加数十年。

### 5.3 夺舍规则（基于原文三大铁则）

原文 1906-1909 行明确规定了夺舍三铁则：

1. **不可夺舍凡人**：夺舍凡人会导致躯体自动崩溃
2. **以强凌弱**：法力高者向法力低者夺舍才可能成功，法力差距越大越安全
3. **一生一次**：每名修仙者一生只可进行一次夺舍，第二次时元神自动消亡

**游戏实现建议**：
- 寿命耗尽前最后一个机会为夺舍
- 夺舍需要寻找合适的修士躯体
- 夺舍后退回到被夺舍者的修为水平，但无瓶颈限制可重修（原文 15770 行）
- 成功夺舍后需数月至一年的闭关才能掌控新躯体（原文 9184 行）
- 夺舍后的修炼速度受新躯体灵根资质影响

### 5.4 寿命消耗的游戏机制

- 每过游戏时间 1 年，剩余寿命 -1
- 突破大境界后刷新寿命上限
- 寿命 < 10 年时触发"大限将至"状态，属性衰减
- 寿命归零时触发角色死亡事件（可夺舍续命或轮回转世）
- 特殊事件（透支潜力、抽髓丸等）可消耗当前寿命换取短期战力爆发

## 六、总结：数值框架汇总

### 核心境界经验曲线

```
境界路径：炼气(1-13层) → 筑基(初/中/后) → 结丹(初/中/后) → 元婴(初/中/后) → 化神 → 炼虚 → 合体 → 大乘 → 渡劫 → 真仙
```

### 灵根系统数值乘数

| 灵根类型 | 修炼速度 | 结丹成功率 | 同阶战力 | 出现概率 |
|---------|---------|-----------|---------|---------|
| 伪灵根 | x0.3 | 趋近 0 | x1.0 | ~80% |
| 真灵根 | x1.0 | x1.0 | x1.0 | ~15% |
| 天灵根 | x2.5 | x∞(自动) | x1.2 | ~0.01% |
| 变异灵根 | x2.5 | x1.0 | x3.0(匹配功法时) | ~0.05% |

### 关键设计决策

1. **只设境界限制，不设固定等级**：LPC MUD 不需要传统 RPG 的等级制，境界本身就是最自然的阶段划分
2. **突破是核心玩法**：突破失败后建议保留部分进度（而非完全重置），降低挫败感
3. **灵根决定流派**：伪灵根推荐走丹修/器修路线；天灵根/变异灵根适合走战斗路线
4. **功法非独占**：玩家可同时修炼多种功法（如主修青元剑诀+辅修大衍诀），受限于修炼时间
5. **寿命作为驱动力**：寿命倒计时是修仙游戏区别于普通 RPG 的核心机制
