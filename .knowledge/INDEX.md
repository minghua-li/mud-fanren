# 凡人修仙传 MUD 知识库索引

本索引列出凡人修仙传 MUD 项目所有业务设计文档。各文档聚焦独立维度，设计决策基于剧本（document/凡人修仙传.txt）提炼。

文件按「总览 → 详细」的层级结构组织在子目录中，`ls .knowledge/` 即可看到实际子目录层级。

## 目录结构一览

```
.knowledge/

├── RELATIONS.md                # 关联关系边表

│   ├── 1A-人界地理.md           # 人界详细（详细展开）
│   └── 1B-灵界地理.md           # 灵界详细（详细展开）
├── cultivation/                # 🧘 修炼体系
│   ├── CULTIVATION_SYSTEM.md   # 境界·灵根·功法总览（总览文档）
│   └── 1C-修仙境界功法.md       # 数值框架与 LPC 实现（详细展开）
├── factions/                   # ⚔️ 势力·角色
│   ├── FACTIONS.md             # 势力体系总览（总览文档）
│   ├── 1D-门派种族声望.md       # 门派/种族/声望 LPC 版（详细展开）
│   └── CHARACTERS.md           # 角色体系（独立文档）
├── items/                      # 💎 物品·经济
│   └── 1E-法宝丹药经济.md       # 法宝/丹药/灵石/符箓/材料（完整版）
├── combat/                     # ⚡ 战斗体系
│   ├── COMBAT_SYSTEM.md        # 法术·剑诀·阵法总览（总览文档）
│   └── 1F-法术剑诀阵法战斗.md   # LPC 详细实现（详细展开）
├── quests/                     # 📜 任务系统
│   └── 1G-任务副本奇遇.md       # 主线/副本/奇遇完整设计（完整版）
└── 02-扩充内容/                # ✨ 合理扩充（游戏化设计）
    ├── 02-经济与资源.md         # 灵石经济、市场物价、日常收支（合理扩充）
    ├── 02-日常玩法.md           # 修行生活、副职系统、PVE/PVP 日常（合理扩充）
    ├── 02-配角与势力.md         # 配角视角支线、势力深度、选择分支（合理扩充）
    ├── 02-丹药体系详解.md       # 丹药分类、配方材料、品级效果、炼制门槛（游戏化扩充）
    ├── 02-法宝与武器图鉴.md     # 法宝分类、境界限制、获取途径、相生相克（游戏化扩充）
    └── 02-功法修炼体系.md       # 功法类型、修炼条件、战力成长、冲突融合（游戏化扩充）
```
