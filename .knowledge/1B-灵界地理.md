# 1B：灵界地理与种族区域设计

基于剧本《凡人修仙传》（忘语著）提炼。本文档聚焦灵界（中界）的地理区域结构、各大种族分布、领地边界与进入门槛。

---

## 总览：灵界格局

灵界是人界修士化神后可飞升的更高位面。灵气浓度远超人界，面积无限广阔，至今无人探清边界。

灵界已知分为**三块大陆**，每一块都幅员辽阔，远超下界所有人界大陆之和：

```
灵界（已知三大陆）
├── 风元大陆（最小，种族最多） — 人族三皇境 + 七妖王地 + 天渊城
│   ├── 天元境（儒皇·天元圣皇统治）
│   ├── 玄武境（霸皇·玄武霸皇统治）
│   ├── 天灵境（灵皇·天妙灵皇统治）
│   └── 七妖王地（七大妖王分管）
│
├── 雷鸣大陆（第二大） — 天云十三族、角蚩族、海王族等群雄割据
│   ├── 天云十三族联盟（万古族、石茧族、黑儒人等）
│   ├── 角蚩族领地
│   ├── 海王族领地
│   └── 其他独立种族（雪吼族、乌罗族、娲氏等）
│
└── 血天大陆（最神秘） — 种族信息极少，以诡异蛊虫闻名
    ├── 血族领地
    └── 未知神秘势力
```

### 灵界核心设定

- **超级仙阵**：初代三皇和七妖王及众多大神通者联手布下，笼罩三皇境和七妖地。设有十个阵眼（人族三境各一、妖族七地各一）。个体越强大越受禁制阻挡，中低阶存在不受限
- **人妖关系**：半敌半友。妖族整体实力略胜人族，但人族高阶修士略强于妖族同阶。面对外族时联手抗敌
- **界面法则**：下界化神修士飞升灵界后，需服三百年「灭尘丹」洗去下界气息，否则三百年一次的两色雷劫将无法渡过
- **天渊城体制**：飞升修士必须在天渊城服役，每服一粒灭尘丹，服役三百年后可自由离开

## 一、风元大陆（人族主大陆）

### 1.1 大陆概况

风元大陆是已知三块大陆中最小的一块，但**种族密度最高**。人族、妖族、灵族、木族、夜叉族、影族等大小六七个种族在此大陆共存。大陆西北角尤为混乱——异族领地交错，冲突不断。

### 1.2 三皇境（人族三大统治区）

| 境域名 | 圣皇 | 称号 | 主城 | 特色 |
|--------|------|------|------|------|
| **天元境** | 天元圣皇 | 儒皇 | 天元城（天元圣山） | 风俗最接近人界；炼体士出身，后改修儒家法门 |
| **玄武境** | 玄武霸皇 | 霸皇 | 玄武城（骖灵古龟背上） | 建在远古巨龟"骖灵"背上，可缓缓移动的海上巨城 |
| **天灵境** | 天妙灵皇 | 灵皇 | 天灵城（擎天灵树） | 建在直径百里的擎天灵树上，传说可直通仙界 |

三片境域各设数百到上千的大小城市。普通城市如安远城可居住近亿人口，主城面积堪比人界一个小国家。

#### 超级仙阵三大阵眼（人族侧）

1. **天元圣山** — 天元城所在
2. **骖灵古龟** — 玄武城的移动基石
3. **擎天灵树** — 天灵城的根基

除非摧毁圣山、古龟、灵树三者之一，否则人类侧禁制安然无恙。

### 1.3 七妖王地

妖族七大妖王各自统治的区域，与人族三皇境相邻。妖族整体实力略胜人族，双方签订了一系列协议：
- 人族高阶修士不得任意清剿低中阶妖兽
- 妖族区域内须专门设立人族聚集区并受保护
- 兽潮被视为自然筛选，两族高层默许其存在

主要妖王：金蛟王等（其余妖王具体名号剧本未全展开）。

### 1.4 天渊城（人妖联军要塞）

**天渊城**是人族与妖族联军驻守的边境要塞，位于超级仙阵的唯一入口，是抵御异族入侵的最前线。

**面积**：天渊城所辖面积相当于人界一个小国家。真正的城区数万里见方，由高大城墙围起。

**结构**：
- **梯形城区**：外城墙高大森严，布满禁制；内墙低矮，连接三境七地
- **一分为二**：城内以巨大光幕将城区从中间一分为二，人族与妖族各管辖一面
- **长老塔**：城中心最高塔，合体期长老议事之所
- **飞灵殿**：飞升修士报到、领取灭尘丹
- **玉阙阁**：处理灵地分配等事务

**守卫体系**（按战甲颜色分阶）：

| 级别 | 战甲 | 修为要求 | 说明 |
|------|------|---------|------|
| 黑铁卫 | 黑色 | 元婴期 | 最低阶卫士 |
| 青冥卫 | 青色 | 化神期 | 小队长级，统率十名黑铁卫 |
| 金庭舟/天渊卫 | 金色 | 炼虚期+ | 高阶存在，可自由穿梭各地区域不受限制 |
| 长老会 | — | 合体期 | 十余名成员，最高决策层 |

**三大灵脉山脉**：

| 山脉名 | 灵气品质 | 说明 |
|--------|---------|------|
| **翠泷山** | 最佳 | 化神修士洞府最多，出产几种特别灵药 |
| **天韵山** | 中等 | 适合化神修士修炼 |
| **琼籁山** | 较低 | 原属元婴修士，后划归化神使用，洞府稀疏、灵地广大 |

**外围区域**：
- **浮黎沼泽**：城外万里处，地形特殊可掩蔽神念扫描，常有异族潜伏
- **凡人之地**：城外有凡人村镇，提供生活物资和低阶修炼者人才

**进出规则**：天渊城面对蛮荒世界的巨墙无城门，进出全靠传送阵。

### 1.5 人族真灵世家

人族中存在古老的真灵世家，血脉传承自远古真灵：

| 世家 | 血脉 | 特色 |
|------|------|------|
| **陇家** | 真龙血脉 | 掌控遍布三境之地的拍卖行 |
| **叶家** | 天凤血脉 | 暗中操控大量极品灵石矿 |
| **谷家** | 推测天凤分支 | 与叶家类似，罕见在公开活动 |

这些世家子弟常隐匿身份在天渊城试炼。

### 1.6 风元大陆的其他区域

**黑隐山脉**：
- 半岛连接风元大陆，大部分被黑冥雾笼罩
- 被飞灵族下的天鹏族统治
- 修士需向天鹏族缴纳供奉（木铃花为首要供奉物）

**火瑚群岛**：
- 风元大陆与雷鸣大陆之间的海域群岛
- 原有传送阵直通雷鸣大陆，后被海兽摧毁
- 娲氏三族在此立足数千年
- 乌罗族残党在此区域活动

## 二、雷鸣大陆（第二大洲）

### 2.1 大陆概况

雷鸣大陆是灵界三块已知大陆中第二大的。此大陆上种族势力极为复杂，以**天云十三族联盟**和**角蚩族**的对抗为主线。大陆西部与风元大陆隔着浩瀚海域，飞渡需大半年之久。

### 2.2 天云十三族联盟

雷鸣大陆最庞大的种族联盟之一，由十三个种族组成，共同对抗角蚩族的入侵。联盟核心城市为**云城**（第三大城）和**金甲城**（要塞）。

| 种族 | 特点 | 排名/地位 |
|------|------|-----------|
| **万古族** | 天云十三族之一，傀儡术造诣极深 | 中等排名 |
| **石茧族** | 擅长炼器，与万古族合作建造云城防御系统 | 中等排名 |
| **黑儒人** | 本身不善飞遁，依赖飞车 | 中等种族 |
| **心魅族** | 小族，有危险感应的天赋神通 | 小族 |
| （其余八族名） | 剧本未详细展开 | — |

**云城（天云十三族第三大城）**：
- 仅次于天空之岛的悬浮居住之所，可随季节升降高度
- 城体遮蔽大半天空，以巨大球体作为防御
- 金属类宝物和材料不得靠近球体千丈之内
- 万古族和石茧族合力建造防御系统

**金甲城**：
- 天云十三族在此区域的最大城市和要塞险关
- 长期驻扎大批精锐部队和高阶存在

### 2.3 角蚩族

雷鸣大陆的超级大族之一，以好战扩张闻名。

**特征**：
- 头生独角（白角是族中显眼标志）
- 拥有飞行战舟，战力强大
- 灭族之战风格——不宣而战，屠城，抓人做奴隶或炼丹炼器
- 拥有冰火黑狱（关押奴隶）

**与天云十三族的战争**：
角蚩族突然发动吞并战争，切断天云十三族的传送阵网络。绿光城被大肆屠戮。被俘者要么投入冰火黑狱为奴，要么被搜魂抽婴用作炼丹炼器材料。

### 2.4 海王族

与角蚩族并称的雷鸣大陆另一超级大族，两族都拥有轻易灭掉飞灵族等中小种族的实力。与角蚩族的关系时而敌对时而联手。

### 2.5 雷鸣大陆其他种族

| 种族 | 特点 | 实力评级 |
|------|------|---------|
| **地灵族** | 雷鸣大陆机关傀儡术第一（已灭绝） | 曾极强 |
| **雪吼族** | 雷鸣大陆极北之地，个体战力不逊大族高阶存在 | 中等但好战 |
| **乌罗族** | 应已灭绝，残党在火瑚群岛活动；王族可克制娲氏 | 难缠 |
| **娲氏** | 雷鸣大陆附近海域的三族之一 | 中等 |
| **戎族** | 出现在广寒界 | 不详 |
| **心魅族** | 小族，天赋感应危险 | 弱小 |
| **黑儒人** | 不擅飞遁 | 中等 |
| **飞灵族** | 带翅种族，数十分支；天鹏族曾是最大分支 | 中等偏弱 |

### 2.6 广寒界（雷鸣大陆专属秘境）

- **位置**：雷鸣大陆上空的神秘空间
- **开启周期**：每万年开启一次
- **进入限制**：圣族以上（大乘期）无法进入
- **令牌**：「广寒令」，每次出世数百块，以各种诡异方式出现在雷鸣大陆各处
- **传送能力**：一块令牌可带十几人到三十多人进入（视各族的传送法阵能力）
- **折损率**：参与的各组圣子折损率约六七成
- **进入者修为**：一般相当于化神级的"飞灵将"级别
- **重要性**：各族实力大增的良机
- **内容**：上古战场遗迹，含密洞、金篆文、练神术等机缘

### 2.7 跨大陆传送阵

天云十三族掌握一座可以跨大陆（雷鸣大陆↔风元大陆）的超级传送阵，但每次开启消耗极大，足以让圣族存在倾家荡产。坐落在伏蛟城密殿中。战争期间会被封锁。

## 三、血天大陆（最神秘的大陆）

### 3.1 大陆概况

血天大陆是灵界三块已知大陆中最神秘的一块，相关信息极少。剧本中仅在后期提及：

- 拥有独特奇虫**血蛊虫**（血天大陆独有），用于下蛊暗算
- 冰魄仙子最后在此大陆失踪
- 危险程度极高，合体期修士也难以全身而退
- 可能涉及血族等特殊种族

> **注**：血天大陆在剧本中的信息量有限，适合作为游戏高级区域或剧情伏笔使用。

### 3.2 已知关联

- 人族修士许家某位前辈（血魂）曾前往血天大陆寻访冰魄下落，中血蛊虫后九死一生逃回风元大陆
- 韩立通过雷鸣大陆典籍了解到血蛊虫的来历
- 血天大陆的势力结构和种族分布剧本中未详细展开

## 五、灵界修炼体系与境界

### 5.1 飞升与天劫

| 阶段 | 说明 |
|------|------|
| **飞升灵界** | 化神期修士飞升后出现在飞灵台，被接往天渊城 |
| **灭尘丹服务期** | 需服三百年灭尘丹洗去下界气息，期间在天渊城服役 |
| **两色雷劫** | 未服灭尘丹的飞升修士遭遇（比普通天劫更凶险） |
| **小天劫** | 元婴以上修士在灵界都需经历，三百年一次 |
| **大天劫** | 合体期修士面临的大劫，有的自知无望会用毕生法力占卜 |

### 5.2 灵界修炼资源

- **灭尘丹**：飞升修士必须，材料需从蛮荒世界获取
- **真蟾液**：对化神后期修士也有效用的珍稀灵液
- **天心丹**：化神期丹药
- **万妙丹**：天云十三族万古族的神丹
- **广寒界灵药**：万年一开，收获珍稀灵药和秘术

## 九、设计要点总结

### 9.1 核心设计原则

## 总览：灵界地理格局

灵界是人界化神以上修士飞升到达的高阶界面，灵气浓度远超人界（约10倍以上），面积无限广阔。灵界共分三境七地，由多位大乘修士（三皇七妖）统治，此外还有一座超然于各方势力的要塞巨城——天渊城。

灵界已知的几大板块：

```
灵界
├── 雷鸣大陆（中央大陆，人族/妖族核心区域）
│   ├── 天渊城（抗击异族的前线要塞）
│   ├── 三皇领地（人族三大势力范围）
│   ├── 七妖之地（妖族七大妖王领地）
│   └── 蛮荒世界（城外未探明的异族/妖兽区域）
├── 血天大陆（异大陆，夜叉/飞灵等族聚居）
│   ├── 夜叉族领域
│   ├── 飞灵族领域
│   └── 血族/角蚩族势力范围
├── 地渊（地下洞窟世界，多层结构）
│   ├── 上层（广寒界入口层）
│   ├── 中层（四大妖王领地）
│   └── 下层（远古战场/封印之地）
├── 广寒界（独立秘境，上古战场遗迹）
└── 其他区域
    ├── 魔金山脉（各族秘境试炼地）
    ├── 碧灵岛（灵界版）
    └── 天沙大陆等边缘地域
```

**关键的界面通路：**
- 人界→灵界：化神期飞升时经「飞灵台」接引，人族飞升者均被接入天渊城
- 灵界内部：古传送阵连接各大陆，或通过蛮荒世界跨境飞行（极其危险）
- 灵界→魔界：界面裂缝/空间通道（魔界入侵通道）
- 灵界→仙界：大乘期渡飞升劫，真仙之门

## 一、雷鸣大陆

### 1.1 总述

雷鸣大陆是灵界已知区域中最核心的大陆，人族和妖族的主要聚居地。大陆幅员辽阔，分为三皇治理的人族区域、七妖王统领的妖族领地、以及被称为「蛮荒世界」的未探明区域。

### 1.2 地图布局与跨区连接

```
蛮荒世界（城西/南/北——三重空间折叠扩散）
    │
    ├── 天渊城（中央要塞，跨大陆传送枢纽）
    │       ├── 人族城区（东半区：巨塔群、坊市、洞府山脉）
    │       ├── 妖族城区（西半区：化形妖族聚居）
    │       └── 长老塔群（中央核心，合体期长老议事）
    │
    ├── 三皇领地（天渊城后方）
    │   ├── 天灵境（人族繁盛区域，三大主城之一天元城）
    │   ├── 万宝境（各种交易中心）
    │   └── 金翰境（修仙宗门聚集地）
    │
    └── 七妖之地（妖族自治区域）
        ├── 天奎狼原（天奎神狼统治）
        ├── 蛟龙渊（蛟龙一族领地）
        └── 其他妖王领地
```

**区域连接：**
- 天渊城↔三皇领地：通过超大型传送阵（天渊城内设传送殿），需缴纳传送费
- 天渊城↔蛮荒世界：只能通过城中传送阵传输，无城门直通
- 天渊城↔雷鸣大陆其他区域：飞行或传送阵均可
- 三皇领地内部：有国级传送阵网络

### 1.3 天渊城（核心枢纽）

天渊城是灵界人族与妖族联合建造的超级要塞，是抗击外族入侵的最前线。与其说是一座城，不如说是一个「国」——所辖面积相当于人界的一个小国家。

**规模与结构：**

| 层级 | 说明 |
|------|------|
| **城区** | 数万里城墙围起的真正城区，呈梯形分布，以光幕分隔人妖两族 |
| **山脉带** | 城区外连绵山脉：翠泷山（灵气最足）、天韵山、琼籁山；修士洞府所在 |
| **凡人村镇** | 山脉间散布的凡人聚落，提供生活物资和低阶修士来源 |

**核心设施：**

| 设施 | 说明 |
|------|------|
| **中心巨塔**（长老塔） | 高达千丈，是其他巨塔的两倍粗；合体期长老议事决断之所 |
| **黄/白/黑/青/金甲塔群** | 按卫士等级分布的驻军巨塔，构成天渊城的主要战力体系 |
| **传送殿** | 连接蛮荒世界和族内各地的核心传送枢纽 |
| **坊市** | 三境七地所有珍稀材料应有尽有，规模极大 |
| **玉阙阁** | 洞府分配管理机构 |
| **天广殿** | 每十年一次合体期长老讲道之地，天渊城核心特权之一 |

**卫士体系：**

| 阶位 | 战甲颜色 | 修为要求 | 职责 |
|------|---------|---------|------|
| **黑铁卫** | 黑色 | 元婴期 | 基础巡逻守卫 |
| **青冥卫** | 青色 | 化神期 | 小队统领，执行普通任务 |
| **天卫（金甲）** | 金色 | 炼虚期及以上 | 高阶执法，可自由穿梭各族区域，斩杀一切威胁 |

**天渊城特殊设定：**
- **灵气浓度：** 天渊城内修炼速度是外界的1倍以上（核心区域可达2倍）
- **灭尘丹：** 飞升修士需连服300年灭尘丹以洗去下界异界气息，否则每300年遭遇两色雷劫
- **飞升修士：** 天渊城高阶修士中，飞升修士占比不到一成，但战力普遍强于本土修士
- **异族入侵：** 每6-10万年一次外族攻城（大规模），中间有零星异族袭扰
- **长老会：** 由十几名合体期修士组成（人妖各半），除金越禅师和雷罗真人外还有多位长老
- **护城仙阵：** 上古遗留的超级阵法，异族入侵时只能从天渊城正面一个入口攻入

### 1.4 三皇领地

人族区域由三皇统治，三位大乘修士各自管理一片广袤领地：

| 三皇 | 领域 | 特点 |
|------|------|------|
| **天皇** | 天灵境 | 人族核心区域，天元城为人族三大主城之首 |
| **地皇** | 万宝境 | 交易中心，各种珍稀资源汇聚 |
| **人皇** | 金翰境 | 修仙宗门聚集，传承最为丰富 |

**三大主城：**
- **天元城**：天灵境首府，规模在人族三大主城中最大
- **其他两座主城**：位置与名称在剧本中未详细展开

**主要修仙家族（真灵世家）：**
- **陇家**：真龙血脉，操控遍布三境的拍卖行
- **叶家/谷家**：天凤血脉，暗中掌握人族极品灵石矿
- 这些真灵世家族人常在化神期时隐藏身份进入天渊城试炼

### 1.5 七妖之地

妖族（化形妖兽）在雷鸣大陆拥有独立的广袤领地，以七大妖王统领：

| 妖王 | 领地 | 原型 |
|------|------|------|
| **天奎神狼** | 天奎狼原 | 天奎狼（与银月相关） |
| **蛟龙妖王** | 蛟龙渊 | 蛟龙 |
| **其余五妖王** | 各自领地 | 剧本未详述 |

**人妖联盟：** 为在灵界残酷的生存竞争中立足，人妖两族缔结永久同盟，共同驻守天渊城。两族平时互不干涉内政，只在异族入侵时统一听从天渊城长老会调遣。

### 1.6 区域灵气等级与怪物分布

| 区域 | 灵气等级 | 怪物/敌对势力 | 建议境界 |
|------|---------|--------------|---------|
| 天渊城城区 | 极高（×2修炼速度） | 无（安全区） | 不限 |
| 天渊城外山脉 | 高 | 低阶妖兽、偶有异族探子 | 元婴~化神 |
| 三皇领地腹地 | 中~高 | 野生妖兽 | 炼虚~合体 |
| 蛮荒世界外围 | 中 | 异族探子、低阶异族 | 化神~炼虚 |
| 蛮荒世界深处 | 高 | 异族大军、真灵级存在 | 合体~大乘 |

### 1.7 进入条件与特殊规则

- **天渊城：** 所有飞升人族自动入城（需服用灭尘丹300年）；本土修士可通过轮值或军方关系加入
- **三皇领地：** 所有灵界人族自由通行（天渊城修士休假期间可前往）
- **蛮荒世界：** 必须从天渊城传送，不允许直接出城；需组队或至少青冥卫级别

## 二、血天大陆

### 2.1 总述

血天大陆是灵界另一块已知的大陆，与雷鸣大陆相隔遥远。这里是夜叉族、飞灵族、血族等非人种族的聚居地。

### 2.2 地图布局与跨区连接

```
血天大陆
├── 夜叉族领域（大陆中部及北部）
│   ├── 夜叉王城（族内核心）
│   └── 外层领地（与其他种族交界地带）
├── 飞灵族领域（大陆东部及南部）
│   ├── 飞灵台（飞升接引台，与灵界各族飞升有关）
│   └── 飞灵族聚居地
├── 血族领地（西部区域）
└── 角蚩族势力范围（与雷鸣大陆接壤处）
```

**跨区连接：**
- 血天大陆↔雷鸣大陆：古传送阵连接（极少数存在且需要大神通驱动）
- 血天大陆内部：各领域间有传送阵和飞行路线
- 血天大陆↔蛮荒世界：直接接壤，边境冲突频繁

### 2.3 主要种族与势力分布

| 种族 | 特点 | 社会形态 |
|------|------|---------|
| **夜叉族** | 灵界血天大陆主要种族，肉身强悍 | 部族联盟制，有夜叉王 |
| **飞灵族** | 背生双翼，擅长飞行与空间神通 | 部落制，有飞灵族族长 |
| **血族** | 擅长血道功法，以血为力量之源 | 氏族制，血祖统领 |
| **角蚩族** | 灵界强族之一，好战侵略性强 | 军事帝国制 |
| **海王族** | 海洋种族，与人族有领土纠纷 | 深海王国体系 |

### 2.4 区域灵气等级与怪物分布

| 区域 | 灵气等级 | 怪物/敌对势力 | 进入条件 |
|------|---------|--------------|---------|
| 夜叉族领域 | 中~高 | 夜叉族战士 | 化神期以上/持有通行许可 |
| 飞灵族领域 | 中 | 飞灵族巡逻队 | 友好声望/化神期以上 |
| 血族领地 | 中 | 血族卫队、血兽 | 炼虚期以上 |
| 角蚩族战区 | 高（但风险极大） | 角蚩族军队 | 合体期以上 |
| 边缘荒原 | 低~中 | 各种异族游荡者、妖兽 | 化神期以上 |

### 2.5 进入条件与特殊规则

- 非本族进入其他种族核心区域需获得对方许可
- 人族与血天大陆各族不存在天渊城式的永久联盟关系——多数为中立或敌对
- 血天大陆各族也面临魔界入侵威胁

## 三、地渊

### 3.1 总述

地渊是灵界的地下洞窟世界，共有七层（或更多），每层都有独特的生态环境和原住种族。这是远古妖兽和异族的栖息地，也是广寒界的入口所在层。

### 3.2 多层洞窟结构

```
地渊七层
├── 第一层（入口层）—— 熔岩与地下森林交错
│   └── 广寒界入口（与地渊第一层相通的上古禁制入口）
├── 第二层（幽暗层）—— 磷光植物、地下河流
├── 第三层（妖兽巢穴层）—— 远古妖兽栖息
├── 第四层（四大妖王领地）—— 地渊中最强的四大妖王统治
├── 第五层（封印层）—— 远古战场遗迹、封印禁制
├── 第六层（幽深未知层）—— 几乎无人探索
└── 第七层（地心层）—— 传说中的最深处，真灵沉睡之地
```

### 3.3 主要势力

| 势力 | 位置 | 说明 |
|------|------|------|
| **四大妖王** | 第四层 | 地渊本地的顶级存在，修为相当于合体~大乘期，各自统治一层区域 |
| **远古妖兽** | 第三~五层 | 灵界已经罕见的远古种，血脉纯正、战力惊人 |
| **异族部落** | 各层碎片分布 | 在灵界无法立足的小种族，退居地渊深处求存 |

### 3.4 区域灵气等级与怪物分布

| 层数 | 灵气等级 | 怪物等级 | 建议境界 |
|------|---------|---------|---------|
| 第一层（入口层） | 中 | 7~8级妖兽（元婴~化神） | 化神期 |
| 第二层（幽暗层） | 中 | 8~9级妖兽（化神~炼虚） | 化神后期~炼虚 |
| 第三层（巢穴层） | 中~高 | 9级妖兽（炼虚） | 炼虚期 |
| 第四层（妖王层） | 高 | 10级+（合体期妖王） | 合体期 |
| 第五层（封印层） | 高 | 远古封印、残余战场气息 | 合体期+ |
| 第六层 | 极高（但环境恶劣） | 未知 | 大乘期 |
| 第七层 | 超高 | 真灵级 | 大乘后期~真仙 |

### 3.5 进入条件与特殊规则

- **入口位置：** 雷鸣大陆或血天大陆的某些特定地域有通往地渊的洞穴入口
- **最低准入：** 化神期（第一层），每深入一层需提升一个大境界
- **特殊资源：** 地渊产出灵界罕见的远古灵药和珍稀矿物
- **死亡惩罚：** 在地渊陨落将极难复活—阴司之界也无法收束某些远古禁制的灵魂

## 五、广寒界

### 5.1 概述

广寒界是灵界已知最重要的一处上古秘境，相传是上古时期某场大战后留下的战场遗迹。广寒界每间隔一定时间开启，各族精英可进入其中寻宝试炼。

### 5.2 地图布局

```
广寒界（独立空间秘境）
├── 外层区域（较安全）
│   ├── 灵气沼泽
│   ├── 废弃传送阵遗址
│   └── 小型遗迹群
├── 中层区域（核心宝物区）
│   ├── 上古洞府（已探索）
│   ├── 灵药园遗址
│   └── 炼器/炼丹室遗迹
└── 内层区域（极度危险）
    ├── 上古战场核心
    ├── 禁制最密集区域
    └── 封印之地（可能封印着远古存在）
```

### 5.3 入口与进入条件

| 项目 | 说明 |
|------|------|
| **入口位置** | 地渊第一层某处（有上古禁制隐藏） |
| **开启周期** | 数百年至数千年一次（具体周期不定） |
| **进入条件** | 化神期以上；需持有特殊「广寒令」或通过特定通道 |
| **进入限制** | 合体期以上无法进入（空间法则排斥高阶修士） |
| **持续时间** | 通常为数年（空间关闭前必须离开） |

### 5.4 区域灵气等级与怪物分布

| 区域 | 灵气等级 | 危险程度 | 产出 |
|------|---------|---------|------|
| **外层区域** | 高 | 中等 | 普通灵药、低阶古宝 |
| **中层区域** | 极高 | 高 | 高阶灵药、古宝、传承 |
| **内层区域** | 超量（甚至狂暴） | 极高 | 玄天之宝级别、上古功法、神秘传承 |

### 5.5 特殊规则

- **修为压制：** 合体期及以上无法进入，但化神~炼虚最佳
- **禁制繁多：** 上古禁制极其玄奥，贸然触碰可能形神俱灭
- **夺宝混战：** 各族精英同场竞争，杀人夺宝是常态
- **空间关闭：** 关闭前不离开则极大概率陨落（被空间法则碾碎）
- **宝物等级：** 广寒界内层可能产出玄天之宝级别甚至更高级的法宝残片

## 八、设计要点总结

### 8.1 核心设计原则

1. **渐进式展开：** 玩家从人界化神飞升→进入天渊城（新手期灵界）→探索三皇领地→进入血天大陆/地渊→广寒界试炼，逐步深入
2. **高门槛准入：** 每个灵界区域对玩家最低境界有严格要求，保证挑战与回报匹配
3. **种族多样性：** 30+种族各有文化、阵营、声望系统，丰富交互选择
4. **战争威胁：** 魔界入侵（大后期事件）和异族攻城是天渊城永恒的主线和驱动力

### 8.2 地图展开顺序（按韩立行历建议）

```
人界飞升 → 天渊城（化神期，三百年服役） 
→ 天渊城周边任务（巡逻/猎妖） 
→ 三皇领地周游（炼虚期探索） 
→ 血天大陆访问（合体期） 
→ 地渊探险（合体期） 
→ 广寒界试炼（炼虚~合体期）
→ 蛮荒世界深处（合体~大乘期） 
→ 魔界战争（大乘期） 
→ 飞升仙界（大乘后期→真仙）
>>>>>>> origin/main
```

## 十、游戏化建议

### 10.1 天渊城服务机制

- 飞升玩家绑定天渊城，每年需领取灭尘丹任务
- 完成巡逻/侦查/采集等任务换取灭尘丹
- 累积300年服务后可自由离开（或可提前买断）

### 10.2 种族声望系统

玩家可通过完成各族任务提升声望，解锁对应区域通行权限、商店和特殊任务：

| 种族 | 声望收益 |
|------|---------|
| 天云十三族 | 使用跨大陆传送阵、购买雷鸣大陆特产 |
| 石茧族 | 高级炼器图纸、特殊材料 |
| 万古族 | 傀儡术传承、万妙丹兑换 |
| 飞灵族 | 飞行类秘术、羽族特有物品 |
| 木族 | （敌对声望路线）深入木族领地的可能 |

### 10.3 广寒界副本设计

- 万年开启一次（游戏内可设计为定期活动）
- 需要广寒令进入
- 参与等级：炼虚期以上
- 产出一：上古修士传承（功法/秘术）
- 产出二：珍稀灵药
- 产出三：金篆文/金阙玉书残页
