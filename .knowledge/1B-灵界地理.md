# 1B：灵界地理与种族区域设计

基于剧本《凡人修仙传》（忘语著）提炼。本文档聚焦灵界（中界）的地理区域结构、各大种族分布、领地边界与进入门槛。

---

## 总览：灵界格局

灵界是人界修士化神后可飞升的更高位面。灵气浓度远超人界，面积无限广阔，至今无人探清边界。

灵界已知分为**三块大陆**，每一块都幅员辽阔，远超下界所有人界大陆之和：

```
灵界（已知三大陆）
├── 风元大陆（最小，种族最多） — 人族三皇境 + 七妖王地 + 天渊城
│   ├── 天元境（儒皇·天元圣皇统治）
│   ├── 玄武境（霸皇·玄武霸皇统治）
│   ├── 天灵境（灵皇·天妙灵皇统治）
│   └── 七妖王地（七大妖王分管）
│
├── 雷鸣大陆（第二大） — 天云十三族、角蚩族、海王族等群雄割据
│   ├── 天云十三族联盟（万古族、石茧族、黑儒人等）
│   ├── 角蚩族领地
│   ├── 海王族领地
│   └── 其他独立种族（雪吼族、乌罗族、娲氏等）
│
└── 血天大陆（最神秘） — 种族信息极少，以诡异蛊虫闻名
    ├── 血族领地
    └── 未知神秘势力
```

### 灵界核心设定

- **超级仙阵**：初代三皇和七妖王及众多大神通者联手布下，笼罩三皇境和七妖地。设有十个阵眼（人族三境各一、妖族七地各一）。个体越强大越受禁制阻挡，中低阶存在不受限
- **人妖关系**：半敌半友。妖族整体实力略胜人族，但人族高阶修士略强于妖族同阶。面对外族时联手抗敌
- **界面法则**：下界化神修士飞升灵界后，需服三百年「灭尘丹」洗去下界气息，否则三百年一次的两色雷劫将无法渡过
- **天渊城体制**：飞升修士必须在天渊城服役，每服一粒灭尘丹，服役三百年后可自由离开

## 一、风元大陆（人族主大陆）

### 1.1 大陆概况

风元大陆是已知三块大陆中最小的一块，但**种族密度最高**。人族、妖族、灵族、木族、夜叉族、影族等大小六七个种族在此大陆共存。大陆西北角尤为混乱——异族领地交错，冲突不断。

### 1.2 三皇境（人族三大统治区）

| 境域名 | 圣皇 | 称号 | 主城 | 特色 |
|--------|------|------|------|------|
| **天元境** | 天元圣皇 | 儒皇 | 天元城（天元圣山） | 风俗最接近人界；炼体士出身，后改修儒家法门 |
| **玄武境** | 玄武霸皇 | 霸皇 | 玄武城（骖灵古龟背上） | 建在远古巨龟"骖灵"背上，可缓缓移动的海上巨城 |
| **天灵境** | 天妙灵皇 | 灵皇 | 天灵城（擎天灵树） | 建在直径百里的擎天灵树上，传说可直通仙界 |

三片境域各设数百到上千的大小城市。普通城市如安远城可居住近亿人口，主城面积堪比人界一个小国家。

#### 超级仙阵三大阵眼（人族侧）

1. **天元圣山** — 天元城所在
2. **骖灵古龟** — 玄武城的移动基石
3. **擎天灵树** — 天灵城的根基

除非摧毁圣山、古龟、灵树三者之一，否则人类侧禁制安然无恙。

### 1.3 七妖王地

妖族七大妖王各自统治的区域，与人族三皇境相邻。妖族整体实力略胜人族，双方签订了一系列协议：
- 人族高阶修士不得任意清剿低中阶妖兽
- 妖族区域内须专门设立人族聚集区并受保护
- 兽潮被视为自然筛选，两族高层默许其存在

主要妖王：金蛟王等（其余妖王具体名号剧本未全展开）。

### 1.4 天渊城（人妖联军要塞）

**天渊城**是人族与妖族联军驻守的边境要塞，位于超级仙阵的唯一入口，是抵御异族入侵的最前线。

**面积**：天渊城所辖面积相当于人界一个小国家。真正的城区数万里见方，由高大城墙围起。

**结构**：
- **梯形城区**：外城墙高大森严，布满禁制；内墙低矮，连接三境七地
- **一分为二**：城内以巨大光幕将城区从中间一分为二，人族与妖族各管辖一面
- **长老塔**：城中心最高塔，合体期长老议事之所
- **飞灵殿**：飞升修士报到、领取灭尘丹
- **玉阙阁**：处理灵地分配等事务

**守卫体系**（按战甲颜色分阶）：

| 级别 | 战甲 | 修为要求 | 说明 |
|------|------|---------|------|
| 黑铁卫 | 黑色 | 元婴期 | 最低阶卫士 |
| 青冥卫 | 青色 | 化神期 | 小队长级，统率十名黑铁卫 |
| 金庭舟/天渊卫 | 金色 | 炼虚期+ | 高阶存在，可自由穿梭各地区域不受限制 |
| 长老会 | — | 合体期 | 十余名成员，最高决策层 |

**三大灵脉山脉**：

| 山脉名 | 灵气品质 | 说明 |
|--------|---------|------|
| **翠泷山** | 最佳 | 化神修士洞府最多，出产几种特别灵药 |
| **天韵山** | 中等 | 适合化神修士修炼 |
| **琼籁山** | 较低 | 原属元婴修士，后划归化神使用，洞府稀疏、灵地广大 |

**外围区域**：
- **浮黎沼泽**：城外万里处，地形特殊可掩蔽神念扫描，常有异族潜伏
- **凡人之地**：城外有凡人村镇，提供生活物资和低阶修炼者人才

**进出规则**：天渊城面对蛮荒世界的巨墙无城门，进出全靠传送阵。

### 1.5 人族真灵世家

人族中存在古老的真灵世家，血脉传承自远古真灵：

| 世家 | 血脉 | 特色 |
|------|------|------|
| **陇家** | 真龙血脉 | 掌控遍布三境之地的拍卖行 |
| **叶家** | 天凤血脉 | 暗中操控大量极品灵石矿 |
| **谷家** | 推测天凤分支 | 与叶家类似，罕见在公开活动 |

这些世家子弟常隐匿身份在天渊城试炼。

### 1.6 风元大陆的其他区域

**黑隐山脉**：
- 半岛连接风元大陆，大部分被黑冥雾笼罩
- 被飞灵族下的天鹏族统治
- 修士需向天鹏族缴纳供奉（木铃花为首要供奉物）

**火瑚群岛**：
- 风元大陆与雷鸣大陆之间的海域群岛
- 原有传送阵直通雷鸣大陆，后被海兽摧毁
- 娲氏三族在此立足数千年
- 乌罗族残党在此区域活动

## 二、雷鸣大陆（第二大洲）

### 2.1 大陆概况

雷鸣大陆是灵界三块已知大陆中第二大的。此大陆上种族势力极为复杂，以**天云十三族联盟**和**角蚩族**的对抗为主线。大陆西部与风元大陆隔着浩瀚海域，飞渡需大半年之久。

### 2.2 天云十三族联盟

雷鸣大陆最庞大的种族联盟之一，由十三个种族组成，共同对抗角蚩族的入侵。联盟核心城市为**云城**（第三大城）和**金甲城**（要塞）。

| 种族 | 特点 | 排名/地位 |
|------|------|-----------|
| **万古族** | 天云十三族之一，傀儡术造诣极深 | 中等排名 |
| **石茧族** | 擅长炼器，与万古族合作建造云城防御系统 | 中等排名 |
| **黑儒人** | 本身不善飞遁，依赖飞车 | 中等种族 |
| **心魅族** | 小族，有危险感应的天赋神通 | 小族 |
| （其余八族名） | 剧本未详细展开 | — |

**云城（天云十三族第三大城）**：
- 仅次于天空之岛的悬浮居住之所，可随季节升降高度
- 城体遮蔽大半天空，以巨大球体作为防御
- 金属类宝物和材料不得靠近球体千丈之内
- 万古族和石茧族合力建造防御系统

**金甲城**：
- 天云十三族在此区域的最大城市和要塞险关
- 长期驻扎大批精锐部队和高阶存在

### 2.3 角蚩族

雷鸣大陆的超级大族之一，以好战扩张闻名。

**特征**：
- 头生独角（白角是族中显眼标志）
- 拥有飞行战舟，战力强大
- 灭族之战风格——不宣而战，屠城，抓人做奴隶或炼丹炼器
- 拥有冰火黑狱（关押奴隶）

**与天云十三族的战争**：
角蚩族突然发动吞并战争，切断天云十三族的传送阵网络。绿光城被大肆屠戮。被俘者要么投入冰火黑狱为奴，要么被搜魂抽婴用作炼丹炼器材料。

### 2.4 海王族

与角蚩族并称的雷鸣大陆另一超级大族，两族都拥有轻易灭掉飞灵族等中小种族的实力。与角蚩族的关系时而敌对时而联手。

### 2.5 雷鸣大陆其他种族

| 种族 | 特点 | 实力评级 |
|------|------|---------|
| **地灵族** | 雷鸣大陆机关傀儡术第一（已灭绝） | 曾极强 |
| **雪吼族** | 雷鸣大陆极北之地，个体战力不逊大族高阶存在 | 中等但好战 |
| **乌罗族** | 应已灭绝，残党在火瑚群岛活动；王族可克制娲氏 | 难缠 |
| **娲氏** | 雷鸣大陆附近海域的三族之一 | 中等 |
| **戎族** | 出现在广寒界 | 不详 |
| **心魅族** | 小族，天赋感应危险 | 弱小 |
| **黑儒人** | 不擅飞遁 | 中等 |
| **飞灵族** | 带翅种族，数十分支；天鹏族曾是最大分支 | 中等偏弱 |

### 2.6 广寒界（雷鸣大陆专属秘境）

- **位置**：雷鸣大陆上空的神秘空间
- **开启周期**：每万年开启一次
- **进入限制**：圣族以上（大乘期）无法进入
- **令牌**：「广寒令」，每次出世数百块，以各种诡异方式出现在雷鸣大陆各处
- **传送能力**：一块令牌可带十几人到三十多人进入（视各族的传送法阵能力）
- **折损率**：参与的各组圣子折损率约六七成
- **进入者修为**：一般相当于化神级的"飞灵将"级别
- **重要性**：各族实力大增的良机
- **内容**：上古战场遗迹，含密洞、金篆文、练神术等机缘

### 2.7 跨大陆传送阵

天云十三族掌握一座可以跨大陆（雷鸣大陆↔风元大陆）的超级传送阵，但每次开启消耗极大，足以让圣族存在倾家荡产。坐落在伏蛟城密殿中。战争期间会被封锁。

## 三、血天大陆（最神秘的大陆）

### 3.1 大陆概况

血天大陆是灵界三块已知大陆中最神秘的一块，相关信息极少。剧本中仅在后期提及：

- 拥有独特奇虫**血蛊虫**（血天大陆独有），用于下蛊暗算
- 冰魄仙子最后在此大陆失踪
- 危险程度极高，合体期修士也难以全身而退
- 可能涉及血族等特殊种族

> **注**：血天大陆在剧本中的信息量有限，适合作为游戏高级区域或剧情伏笔使用。

### 3.2 已知关联

- 人族修士许家某位前辈（血魂）曾前往血天大陆寻访冰魄下落，中血蛊虫后九死一生逃回风元大陆
- 韩立通过雷鸣大陆典籍了解到血蛊虫的来历
- 血天大陆的势力结构和种族分布剧本中未详细展开

## 五、灵界修炼体系与境界

### 5.1 飞升与天劫

| 阶段 | 说明 |
|------|------|
| **飞升灵界** | 化神期修士飞升后出现在飞灵台，被接往天渊城 |
| **灭尘丹服务期** | 需服三百年灭尘丹洗去下界气息，期间在天渊城服役 |
| **两色雷劫** | 未服灭尘丹的飞升修士遭遇（比普通天劫更凶险） |
| **小天劫** | 元婴以上修士在灵界都需经历，三百年一次 |
| **大天劫** | 合体期修士面临的大劫，有的自知无望会用毕生法力占卜 |

### 5.2 灵界修炼资源

- **灭尘丹**：飞升修士必须，材料需从蛮荒世界获取
- **真蟾液**：对化神后期修士也有效用的珍稀灵液
- **天心丹**：化神期丹药
- **万妙丹**：天云十三族万古族的神丹
- **广寒界灵药**：万年一开，收获珍稀灵药和秘术

| **伏蛟城** | 天云十三族 | 密城 | 超级传送阵所在（雷鸣↔风元） |
| **绿光城** | 天云十三族 | 城市 | 被角蚩族攻破屠城 |

### 6.4 秘境与特殊地域

| 地点 | 所属大陆 | 进入门槛 | 说明 |
|------|---------|---------|------|
| **广寒界** | 雷鸣大陆上空 | 圣族以下，需广寒令 | 万年一开的上古秘境 |
| **浮黎沼泽** | 风元大陆·天渊城外 | 化神期 | 异族探子出没的高危区 |
| **黑叶森林** | 风元大陆·木族 | 炼虚期 | 木族边缘领地，有梦罗树监测 |
| **南艮大沙漠** | 风元大陆 | 炼虚期 | 通向木族的通路 |
| **魔金山脉** | 灵界某处 | 各族争夺 | 秘境试炼地 |
| **黑隐山脉** | 风元大陆西部 | 化神期 | 被黑冥雾笼罩，天鹏族统治 |
| **火瑚群岛** | 风元↔雷鸣之间 | 化神期 | 海域群岛，与世隔绝数千年 |

## 七、区域连接通路

### 7.1 飞升通道

| 起点 | 终点 | 方式 | 条件 |
|------|------|------|------|
| 人界 | 灵界·飞灵台 | 飞升接引 | 化神期修为 |
| 飞灵台 | 天渊城 | 专人接引 | 自动接引 |

### 7.2 跨大陆交通

| 起点 | 终点 | 方式 | 条件 |
|------|------|------|------|
| 风元大陆 | 雷鸣大陆 | 跨大陆超级传送阵 | 消耗极大（圣族级都感吃力） |
| 雷鸣大陆 | 风元大陆 | 跨大陆超级传送阵 | 伏蛟城密殿→风元大陆偏僻处 |
| 火瑚群岛 | 雷鸣大陆 | 传送阵（已毁） | 需空云晶修复 |
| 各大陆间 | 其他大陆 | 直接飞行 | 动辄数年~大半年；深海极度危险 |

### 7.3 大陆内部交通

- **天渊城内**：传送阵为主，城区可飞行
- **天云十三族区域**：传送阵网络（角蚩族入侵时被切断）
- **人族三境**：城市间有传送阵；野外需飞遁
- **远距离**：金庭舟等大型运输灵舟

## 八、魔界

### 8.1 魔界概况

古魔界是与灵界接壤的异位面，魔族的入侵是灵界后期主线。

| 等级体系 | 对应灵界境界 |
|---------|-------------|
| 低阶魔族 | 元婴及以下 |
| 魔将 | 化神 |
| 魔帅 | 炼虚 |
| 魔王 | 合体 |
| 魔尊 | 大乘初期 |
| **圣祖** | **大乘期** |
| **真魔始祖** | **真仙级** |

> 注：圣祖在魔界地位极高，总数稀少，实力极恐怖。元刹圣祖在魔界所有圣祖中排前五。

### 8.2 魔界与灵界的关系

- 灵界（尤其天渊城）是人妖两族抵御魔族入侵的最前线
- 魔界曾与灵界发生过大规模战争
- 部分人类修士修炼上古魔功到极致会飞升魔界而非灵界
- 真魔气可污秽灵脉、将人界转化为魔界环境

## 九、设计要点总结

### 9.1 核心设计原则

1. **三大陆渐进展开**：玩家飞升灵界后先接触风元大陆（天渊城）→ 通过跨大陆传送阵探索雷鸣大陆 → 后期涉足血天大陆
2. **种族关系复杂**：各大陆种族之间既有联盟又有敌对的动态关系网络
3. **境界门槛清晰**：化神期（灵界起点）→ 炼虚期（自由行动）→ 合体期（高层战力）→ 大乘期（圣族级）
4. **天渊城为核心**：作为飞升修士的起点和中转站，绑定灭尘丹服务机制
5. **广寒界为重要副本**：万年一开，种族精英争夺

### 9.2 地图展开顺序（按韩立行历）

```
飞升灵界（天元境·安远城附近）
→ 天渊城（服役三百年，服灭尘丹）
→ 天渊城周边任务（木族任务→黑叶森林）
→ 遭遇夜叉族大军入侵
→ 意外传送到雷鸣大陆（火瑚群岛→角蚩族战区）
→ 天云十三族·云城（广寒界试炼）
→ 超级传送阵返回风元大陆
→ 黑隐山脉（飞灵族·天鹏族区域）
→ 跨大陆传送回人族三皇境
→ 后期：血天大陆探索（冰魄线索）
```

### 9.3 进入门槛建议（游戏化）

| 境界段 | 可开放区域 |
|--------|-----------|
| 化神期（初入灵界） | 天元境三皇境、天渊城、周边安全区 |
| 化神中期~后期 | 天渊城任务（蛮荒世界边缘、浮黎沼泽巡查） |
| 炼虚期 | 木族任务深入、天渊城自由活动、雷鸣大陆初步探索 |
| 炼虚中期~后期 | 广寒界试炼、天云十三族区域、角蚩族战区 |
| 合体期 | 各大陆自由往来、魔族前线、真灵世家核心区域 |
| 大乘期（圣族级） | 魔界通道、灵界蛮荒世界深处、血天大陆核心 |

### 9.4 势力关系简图

```
人族三皇 ──半敌半友── 七妖王
    │                       │
    ├──友好── 天云十三族      ├──敌对── 角蚩族
    │                       │
    ├──敌对── 木族+影族联军    ├──不定── 海王族
    │                       │
    ├──死敌── 夜叉族          ├──不定── 飞灵族
    │                       │
    ├──中立── 灵族            └──被灭── 地灵族
    │
    ├──死敌── 魔族（魔界）

```

## 十、游戏化建议

### 10.1 天渊城服务机制

- 飞升玩家绑定天渊城，每年需领取灭尘丹任务
- 完成巡逻/侦查/采集等任务换取灭尘丹
- 累积300年服务后可自由离开（或可提前买断）

### 10.2 种族声望系统

玩家可通过完成各族任务提升声望，解锁对应区域通行权限、商店和特殊任务：

| 种族 | 声望收益 |
|------|---------|
| 天云十三族 | 使用跨大陆传送阵、购买雷鸣大陆特产 |
| 石茧族 | 高级炼器图纸、特殊材料 |
| 万古族 | 傀儡术传承、万妙丹兑换 |
| 飞灵族 | 飞行类秘术、羽族特有物品 |
| 木族 | （敌对声望路线）深入木族领地的可能 |

### 10.3 广寒界副本设计

- 万年开启一次（游戏内可设计为定期活动）
- 需要广寒令进入
- 参与等级：炼虚期以上
- 产出一：上古修士传承（功法/秘术）
- 产出二：珍稀灵药
- 产出三：金篆文/金阙玉书残页
