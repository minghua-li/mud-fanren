# 物品与经济体系设计（ITEMS_AND_ECONOMY）

## 一、货币体系

### 灵石（修仙界通用货币）

| 等级 | 价值 | 说明 |
|------|------|------|
| **下品灵石** | 基础单位 | 日常交易使用 |
| **中品灵石** | 1 : 100（下品） | 结丹期及以上常用 |
| **上品灵石** | 1 : 100（中品） | 元婴期可持有 |
| **极品灵石** | 可遇不可求 | 一个矿脉可能才出一块，蕴含巨量灵气 |

### 凡间货币
- 金银铜钱：与灵石不直接挂钩
- 玩家早期在凡人江湖使用

### 交易设施

| 类型 | 可交易物 | 分布 |
|------|---------|------|
| **坊市（修士摆摊）** | 灵药、符箓、法器、材料 | 各大修仙城镇 |
| **店铺**（万宝楼等） | 高阶法宝、成品 | 核心区域 |
| **拍卖会** | 稀有物品 | 定期事件 |
| **交换会** | 以物易物 | 太南小会等 |
| **黑市** | 禁物、消息 | 暗处 |

## 二、核心法宝/物品体系

### 掌天瓶（九彩造化瓶）
- **品级**：玄天之宝（接近仙器）
- **功能**：产生绿色灵液，催熟灵药（最核心功能）；后期吸收银色光点→时间法则相关能力
- **设计**：游戏中最顶级的神器级物品，可通过任务链逐步解锁其全部功能
- **不是消耗品**—主角持有到终局

### 青竹蜂云剑
- 韩立本命飞剑
- 从6口筑基期→12口→36口→72口→最终化为玄天斩灵剑
- 剑阵系统（无形剑气→大庚剑阵→青蟠剑阵→元合五极山）
- 游戏中的核心剑修养成装备

### 虚天鼎
- 乱星海传闻的第一秘宝
- 内含补天丹（碎丹成婴关键）

### 其他重要法宝/物品

| 物品 | 品级 | 效果 | 持有者 |
|------|------|------|--------|
| 补天丹 | 奇丹 | 提高结婴成功率 | 虚天鼎产出 |
| 万年灵乳 | 天材地宝 | 法力瞬间回满 | 稀世珍宝 |
| 降尘丹 | 灵丹 | 辅助元婴突破化神 | — |
| 大挪移令 | 特殊道具 | 激活古传送阵 | — |
| 符宝 | 消耗品 | 封印法宝部分威能 | 各境界都能用 |
| 玄天之宝 | 顶级 | 蕴含法则之力 | 各族大乘争夺 |

### 丹药炼制体系

```
灵药采集 → 药性搭配 → 炼丹（需要丹炉/火候/丹方）→ 成丹（品质分上中下）
```

主要丹药：
- **筑基丹** → 筑基期必备
- **凝液丹** → 炼气期提升修为
- **结金丹** → 辅助结丹
- **元婴丹** → 辅助结婴
- **化神丹** → 冲刺化神

### 符箓体系

| 等级 | 示例 | 使用者 |
|------|------|--------|
| 初级下阶符 | 冰弹符、火弹符 | 炼气期 |
| 初级中阶符 | 连珠雷符 | 筑基期 |
| 初级上阶符 | 遁地符、隐身符 | 筑基/结丹 |

| 高级符 | 融灵符、大挪移符 | 化神+ |

### 傀儡体系

- 低阶傀儡（结丹以下）→ 机关守卫
- 高阶傀儡（元婴-合体）→ 人形战斗单位
- 真仙级傀儡（蟹道人）→ 顶级战力
- 天罡血雷/元磁山傀儡 → 特殊系列

## 四、经济系统设计

### 经济循环
1. 打怪/采集 → 原材料 → 加工（炼器/炼丹/制符）→ 成品 → 自用或售卖
2. 灵石消耗途径：修炼（聚灵阵）、疗伤、购买资源、传送费、门派贡献
3. 灵石产出途径：任务奖励、怪物掉落、倒卖物品、开采灵矿

### 物价梯度
- 炼气期流通物品：下品灵石结算
- 筑基期物品：下品~中品灵石
- 结丹期物品：中品~上品灵石
- 元婴+物品：上品灵石/以物易物

### 经济风险
- 被高等级玩家/怪物击杀掉落
- 黑市交易被骗/被抢
- 拍卖会溢价
