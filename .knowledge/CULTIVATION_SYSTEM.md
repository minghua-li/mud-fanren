# 修仙体系设计（CULTIVATION_SYSTEM）

## 一、境界体系

### 下境界（人界阶段）

| 境界 | 寿命 | 核心突破条件 | 设计说明 |
|------|------|------------|---------|
| **炼气期**（1-13层） | ~100年 | 感应灵气→打通经脉→筑基丹结筑基 | 新手阶段，1-3层入门，4-6层初成，7-9层中阶，10-13层高阶 |
| **筑基期**（初期/中期/后期） | ~200年 | 结丹需凝结金丹（真灵根可无瓶颈/天灵根豁免瓶颈） | MUD中的主要游戏阶段之一 |
| **结丹期**（初期/中期/后期） | ~400-500年 | 碎丹成婴，需经历心魔劫 | 战力质变节点 |
| **元婴期**（初期/中期/后期） | ~1000+年 | 元婴出窍→化神，需天地灵气和感悟 | 元婴不灭则不死 |
| **化神期** | ~2000+年 | 被界面排斥→飞升灵界 | 人界天花板 |

### 中境界（灵界阶段）

| 境界 | 寿命 | 核心突破条件 |
|------|------|------------|
| **炼虚期** | ~5000+年 | 化神后吸纳灵界灵气，合五行之气 |
| **合体期** | ~10000+年 | 法体合一，神与身合 |
| **大乘期** | ~数万年 | 积累法力→渡大天劫（每万年一次）→渡飞升劫 |

### 上境界（游戏终局）

| 境界 | 说明 |
|------|------|
| **渡劫期** | 大乘大圆满后引发飞升天劫 |
| **真仙** | 渡过飞升劫后飞升仙界 |
| 金仙/太乙/大罗/道祖 | 仙界高端境界（小说续传相关，基础本MUD可暂不涉及） |

## 二、灵根系统

### 灵根类型

| 类型 | 属性数 | 修炼速度 | 结丹瓶颈 | 出现概率 |
|------|--------|---------|---------|---------|
| **伪灵根** | 4-5种 | 极慢 | 极难突破 | 修仙者中绝大多数 |
| **真灵根** | 2-3种 | 正常 | 正常瓶颈 | 有灵根者中约~20% |
| **天灵根** | 1种（金/木/水/火/土） | 2-3倍速 | 无瓶颈（自动结丹） | 数百年一遇 |
| **变异灵根** | 变异属性 | 接近天灵根 | 正常但法力更强 | 每二三十年出现一个 |

### 变异灵根类型

| 变异灵根 | 基础组合 | 特点 |
|---------|---------|------|
| **雷灵根** | 金+水变异 | 攻击力最强，克制妖魔 |
| **冰灵根** | 水+土变异 | 冰冻控制，防御力强 |
| **风灵根** | 基础属性变异 | 速度极快，身法优异 |
| **暗灵根** | 特殊变异 | 隐匿、吞噬类能力 |

### 特殊体质（与灵根并行）

| 体质 | 特点 | 代表作 |
|------|------|-------|
| 冰凤之体 | 修炼冰属性功法极快 | 冰凤 |
| 七星月体 | 修炼月属性功法有加成 | 银月（玲珑） |
| 元磁之体 | 对元磁类功法特殊亲和 | 海大少 |
| 天阴之体 | 阴属性功法的绝佳载体 | — |

### 游戏设计要点

1. **玩家灵根在角色创建时决定**：建议用 roll 点机制 + 可选固定（伪灵根给普通玩家，天灵根作为稀有奖励）
2. **灵根影响**：修炼速度、可学功法属性、突破概率、特殊事件触发
3. **后天改变**：通过特殊机缘（如洗灵池、净灵莲）可洗练灵根提升品质

## 三、功法体系

### 功法分类

**按属性：** 金/木/水/火/土 + 变异（雷/冰/风/暗/月）

**按品阶：**
- **基础功法**（炼气-筑基可用）：长春功、基础五行诀
- **中等功法**（结丹-元婴可用）：青元剑诀、大衍诀
- **高阶功法**（化神-炼虚可用）：明王诀、梵圣真魔功
- **顶级功法**（合体以上）：五藏锻元功、炼神术、涅槃圣体

**按系别：**
- **剑修系**：青元剑诀（韩立核心功法，从结丹用到飞升）
- **法修系**：各类五行法术
- **体修系**：梵圣真魔功（法体双修）、涅槃圣体、万毒混元身
- **辅助系**：大衍诀（神识增强）、炼神术（神识分化、控制傀儡）

### 技能系统（LPC技能设计）

每门功法包含：
- 被动属性加成（攻击/防御/速度等）
- 主动技能（法术/剑诀/秘术）
- 绝招（每个功法1-3个终结技）

### 核心功法设计

1. **长春功** → 基础木系功法，韩立起点
2. **青元剑诀** → 13层剑诀+6柄青竹蜂云剑→72口飞剑剑阵→紫青双剑→玄天斩灵剑
3. **大衍诀** → 神识修炼功法，可分神控制多目标
4. **梵圣真魔功** → 法体双修，三头六臂魔神法相
5. **明王诀** → 防御型功法
6. **炼神术** → 顶级神识功法，第三层可媲美真仙神识

## 四、瓶颈与突破

每层大境界之间有瓶颈，突破方式：
1. **丹药辅助**：筑基丹、凝液丹、结金丹、元婴丹、化神丹…
2. **机缘突破**：秘境历练、高人灌顶
3. **战斗中突破**：生死一线间激发潜力
4. **积累突破**：水到渠成（天灵根专属优势）

## 五、战斗要素

| 要素 | 说明 |
|------|------|
| **神识** | 探查范围、控物数量、抵挡夺舍 |
| **法力** | 施法消耗，境界越高法力越雄厚 |
| **真元** | 化神及以上境界的核心能量 |
| **灵力** | 攻击/防御的基础驱动源 |
| **法则之力** | 大乘/真仙级理解天地法则后的力量 |

## 六、寿命系统

游戏内应实现寿命倒计时：
- 每过游戏时间1年，剩余寿命-1
- 突破大境界增加寿命上限
- 寿命耗尽角色死亡（可转世轮回或夺舍续命）
- 特殊灵药（万年灵乳等）微量延寿
