## 三、任务设计原则

### 任务类型

| 类型 | 设计要点 | 适合境界 |
|------|---------|---------|
| **跑腿/送信** | 熟悉地图 | 炼气-筑基 |
| **采集/狩猎** | 材料收集 | 全境界 |
| **寻宝** | 探索激励 | 全境界 |
| **护送** | 保护NPC、AI战斗 | 筑基-结丹 |
| **灭杀** | 击杀特定怪物/NPC | 全境界 |
| **阵营/门派战争** | 大规模PVP+PVE | 结丹+ |
| **护送飞升** | 化神以上专属 | 化神+ |

### 任务链
- 主线任务链不可跳过，但可并行分支
- 支线任务链有取舍，影响声望
- 部分任务限时/限境界

## 四、声望与权限

| 声望等级 | 解锁内容 |
|---------|---------|
| 陌生 | 基础交谈 |
| 友好 | 购买普通物品、接基础任务 |
| 信任 | 购买限定物品、接专属任务 |
| 尊敬 | 拜师、学高阶功法 |
| 崇拜 | 获得该势力最高级支持 |
