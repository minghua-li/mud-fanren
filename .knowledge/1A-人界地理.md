# 1A：人界地理与区域设计

基于剧本《凡人修仙传》（忘语著）提炼。本文档聚焦人界（凡间界）的地理区域结构、地标、进入门槛与连接通路。

---

## 总览：人界地理格局

人界以天南地区为中心展开故事。天南并非整个下界，而只是人界已知的一块大陆区域。按已知信息，人界地理可分为以下几大板块：

```
天南地区（越国/元武国/天罗国/风都国/九国盟等）
  ├── 越国（核心舞台，第一章~第三卷）
  ├── 元武国（越国北邻）
  ├── 天罗国（魔道六宗老巢，西部大国）
  ├── 风都国（正道盟大本营，东部大国）
  ├── 九国盟（南部九国联盟，抵御慕兰法士）
  └── 姜国/车骑国（越国与天罗国之间的小国，最先被魔道吞并）

慕兰大草原（九国盟以南）
  └── 草原尽头通往大晋帝国

大晋帝国（人界第一大国，天南十倍之大）

乱星海（海外群岛，独立于大陆的修仙世界）
  ├── 内星海（已探明海域，星宫统治）
  └── 外星海（妖海，未探明妖兽海域）

无边海（天南北极，神仙难飞）
```

---

## 一、越国（天南七国之一）

越国是天南地区的核心舞台，韩立在此度过了从凡人到筑基、结丹的漫长岁月。
越国共分十三个州，七大门派割据。

### 1.1 越国十三州

越国十三州各有特色，剧本中有明确提及的包括：

| 州名 | 位置 | 特点 |
|------|------|------|
| **镜州** | 越国南部 | 七玄门所在的江湖世界，彩霞山（落日峰）在此州 |
| **岚州** | 越国南部 | 面积第八大、富足程度第二，水网密布，产粮大区，嘉元城为岚州第一大城 |
| **建州** | 越国北部 | 面积十三州中第二，多山川丘陵，太岳山脉横贯此州，黄枫谷在此 |
| **溪州** | 越国西部 | 越国最小州郡，四分之三为黄土高坡/沙漠，人口十余万，无门派驻扎 |
| **辛州** | 越国 | 富足程度第一（仅提及排名） |
| **越州** | 越国 | 韩立故乡所在州，荒山野岭丘陵山地为主 |

### 1.2 越国七大门派

越国七大门派是支撑整个越国修仙界的参天大树：

| 门派 | 实力排名 | 特点 | 驻地 |
|------|---------|------|------|
| **掩月宗** | 最强 | 提倡双修之术，女弟子多为容貌上等，以天月神舟为标志 | 越国某地 |
| **灵兽山** | 第二 | 擅长驱兽役虫，弟子服饰花哨，皮囊口袋众多 | 越国某地 |
| **黄枫谷** | 中等 | 韩立所在门派，山门在建州太岳山脉中 | 建州太岳山脉 |
| **清虚门** | 中等 | 道门风格 | 越国某地 |
| **化刀坞** | 中等 | 擅长刀法 | 越国某地 |
| **天阙堡** | 中等 | 弟子服饰以黄色巨印为标志 | 越国某地 |
| **巨剑门** | 中等 | 全为男子，黑衣，背负一人高无鞘巨剑 | 越国某地 |

**门派实力描述：** 越国七派中掩月宗实力最强，灵兽山紧随其后，其余五派实力相近。七派共同把持越国修仙界资源。

#### 黄枫谷（韩立师门）

- **位置：** 建州西部，太岳山脉中部
- **规模：** 弟子一万余人（炼气期>90%、筑基期数百人、结丹期寥寥数人、元婴期一人）
- **山门结构：**
  - 议事大殿（数十米高石殿，三层大门）
  - 传功阁（授功法）
  - 百药园（韩立被分配看守的药园）
  - 岳麓殿（法器、丹药配方、密术收藏处，禁制繁多）
  - 迎宾楼
  - 坊市（太岳山脉东北边缘，与元武国交界处）
- **结丹期修士：** 红拂（唯一女姓结丹）、李化元（韩立师父）等
- **元婴期修士：** 令狐老祖，九百余岁，常年云游在外

#### 掩月宗

- 七大派实力最强
- 提倡双修之术，弟子以女修为主，容貌上等
- 标志性法器：天月神舟
- 擅长魅惑之术

#### 灵兽山

- 实力仅次于掩月宗
- 擅长驱兽、役虫
- 后来实为魔道六宗卧底，在正魔大战中倒戈

### 1.3 太岳山脉（核心区域地图）

太岳山脉位于建州西部，方圆连绵数千里，原始山林、人迹罕至。

```
元武国
  |
  |（北）
  |
太岳山脉 ──── 黄枫谷（山脉中部）
  |         ├── 百药园（南部偏远山谷）
  |         ├── 坊市（东北边缘）
  |         └── 弟子洞府群
  |
  |（南）
  |
越国其他州
```

**关键方位：**
- 黄枫谷坊市：太岳山脉东北边缘，向北百余里即元武国
- 向西二百里 → 溪州边界
- 灵矿（韩立曾被派往守卫）：太岳山脉某处

### 1.4 血色禁地（血禁试炼）

- **位置：** 建州境内某处
- **性质：** 被风属性古禁封闭的秘境，是越国筑基丹主药的唯一来源
- **周期：** 每五年开启一次，七大派各派低阶弟子进入采集
- **进入门槛：** 筑基期以下（炼气期）
- **死亡率：** ≈75%（即四分之三的进入者无法活着出来）
- **入口开启方式：** 由七派结丹期修士联手破禁（用巨剑门石剑测试禁制强度，待禁制减弱时合力开启）

### 1.5 正魔大战主战场——金鼓原

- **位置：** 越国与车骑国交界处
- **说明：** 魔道六宗入侵后，七派与六宗在此形成对峙。每月大打、每日小打，持续消耗战
- **重要程度：** 越国修仙界存亡之战的核心战场

### 1.6 越都

- 越国京城，凡人王朝的中心
- 七派暗中扶持的凡人势力所在地

## 二、岚州与太南谷

### 2.1 岚州

- **地理位置：** 越国南部
- **面积排名：** 越国十三州中第八
- **富足程度：** 仅次于辛州，位列第二
- **自然特征：** 水网密布，湖泊、水道、运河众多；风调雨顺，产粮大区
- **江湖势力：** 岚州三大霸主——惊蛟会（原最强）、独霸山庄、五色门
- **主要城市：** 嘉元城（岚州第一大城、水运枢纽、惊蛟会总舵所在）

### 2.2 嘉元城

- 贯穿越国南北的乡鲁大运河穿城而过
- 水运枢纽，商贸要道
- 韩立曾在此处理墨府恩怨、结识墨氏三姐妹
- 太南谷升仙大会的起点

### 2.3 太南山与太南谷

- **位置：** 岚州最南部，广贵城西四十里处
- **太南山高度：** 三千多米，岚州第四高山
- **太南寺：** 山顶寺庙，以占卜问签灵验著称
- **太南谷：** 修仙者聚集地，常年被浓雾笼罩，凡人无法进入
- **入口：** 太南山北面一个神秘山坡，常年被浓浓白雾笼罩

### 2.4 太南小会（坊市）

- 小型修仙者交易会
- 韩立在此结识青颜真人、购买金竺笔等法器
- 散修、小家族修士聚集

### 2.5 升仙大会

- **性质：** 越国十年一次的修仙者选拔盛会
- **参与条件：** 炼气期修士
- **形式：** 七座擂台（代表七大派），两两胜进制，每座擂台最后十名胜者成为该派新入门弟子
- **奖励：** 入门弟子可获得服食筑基丹资格
- **升仙令：** 各派发给有功修仙家族的特殊令牌，持令可免试入门并获筑基丹

## 三、凡人江湖区域

### 3.1 七玄门（韩立起点）

- **全称：** 七玄门（又称七绝门）
- **创始人：** 七绝上人（二百年前）
- **鼎盛时期：** 曾雄霸镜州数十载，渗透数州，威震越国
- **现状：** 已衰落，迁至镜州彩霞山，控制青牛镇等十几个小城镇
- **规模：** 门下弟子三四千人
- **组织结构：**
  - 外门四堂：飞鸟堂、聚宝堂、四海堂、外刃堂
  - 内门四堂：百锻堂、七绝堂、供奉堂、血刃堂
  - 长老会（与门主并驾齐驱）
- **彩霞山（总舵）：**
  - 原名落凤山，相传五色彩凤落于此化成
  - 镜州境内第二大山，方圆十几里
  - 主峰"落日峰"，从山底到峰顶只有一条路，设十三处哨卡
  - 拥有大大小小十几个山峰，各分堂分别占据
- **神手谷：** 墨大夫住所，有练功静室
- **炼骨崖：** 入门测试地点（竹林→岩壁→山崖）
- **百莽山：** 镜州第一大山

### 3.2 野狼帮

- 七玄门在镜州境内的死对头
- 前身是马贼，被官府围剿招安后残余势力组成
- 凶狠嗜血、敢杀敢拼
- 控制乡镇较多但不会经营

### 3.3 青牛镇

- 七玄门控制下的小镇子
- 韩立入七玄门前在此停留
- 有春香酒楼（韩立三叔做掌柜）

### 3.4 韩立故里

- 越国越州某山边小村
- 韩立出生地，家庭贫困

### 3.5 镜州

- 越国十三州之一
- 七玄门势力范围所在
- 首府镜州城（七玄门已被挤走）

### 4.2 天南四大势力

正魔大战后，天南修仙界形成四大势力格局：

1. **正道盟** — 以风都国为中心的正道联盟
2. **魔道六宗** — 以天罗国为中心的魔道势力（鬼灵门、合欢宗、御灵宗等）
3. **九国盟** — 南部九国联盟，抵御慕兰法士
4. **天道盟** — 由剩余中小国家组成

### 4.3 魔道六宗

天罗国魔道六宗详情：

| 宗门 | 特点 |
|------|------|
| **鬼灵门** | 擅长驱鬼役妖、毒术、暗术，六宗中最弱但实力仍远超掩月宗 |
| **合欢宗** | 擅长双修功法 |
| **御灵宗** | 擅长御兽之术，与灵兽山有隐秘联系 |
| （其余三宗名） | 剧本中未详细展开 |

### 4.4 元武国

- 与越国接壤的中等国家
- 正魔两派实力相当
- 三大正派：**天星宗**（阵法闻名）、**神兵门**（炼器闻名）、**万妙观**（符道闻名）
- 蟠龙江险滩（李化元曾在附近采集某物）
- 付家（元武国最大修仙家族之一，与韩立结仇）
- 齐家（齐云霄的家族，已逐出）

## 五、九国盟与慕兰草原

### 5.1 九国盟

- **位置：** 天南大陆南部，越国往南穿过紫金国等五六个国家
- **性质：** 九个国家组成的紧密联盟
- **目的：** 共同抵御慕兰族法士入侵
- **核心国家：** 虞国（阗天城所在）等
- **阗天城：** 天南唯一的修士之城，九国盟的交易中心
- **历史：** 原越国七派败退后被九国盟吸收，韩立师父李化元战死于法士争斗

### 5.2 慕兰大草原

- **位置：** 九国盟以南，绵延万里
- **面积与跨度：** 从草原一头游牧到另一头需整整五个春秋
- **环境：** 修炼资源极其贫乏
- **草原尽头：** 传说是大晋帝国（天南地区加起来不足其十分之一大）

### 5.3 慕兰族人

- 纯粹的游牧民族，民风彪悍
- 由成千上万的大小部落组成（小的数十万人口，大的数千万部众）
- 燕族（越国/元武国人族）的死敌
- 内部有世敌"突兀人"

### 5.4 法士体系

慕兰族修仙者的独特体系：

- 不称"修士"，尊称"**法士**"
- 境界划分与修士类似
- **核心特色"灵术"：** 结丹前不重法器和符箓，将五行道术修炼得出神入化
- 结合草原土生秘术，形成独特的"灵术"流派
- 同阶法士可稳胜普通修士
- 高阶法士配合法宝后更强

### 5.5 九国盟与法士的战场

- 九国盟与慕兰草原之间隔有近万里的黄土野地
- 草木稀少，常年黄土飞扬，狂风不止
- 在此地战死的修仙者数以万计

### 5.6 大晋帝国（未直接出场，但有重大影响）

通过慕兰草原尽头或古传送阵可到达。

- **面积：** 天南地区不足其十分之一大
- **地位：** 人界第一大国，修士圣地
- **修仙宗门：** 多如牛毛，功法宝物远超天南修士
- **十大魔宗：** 包括阴罗宗（擅长鬼罗幡，收集生魂）等
- **正道宗门：** 天符门（华云州末流小派，以降灵符著称）、四海真院等
- **描述：** "道友若去了我们大晋，才会知道什么叫做真正的修仙界"

## 六、乱星海（海外世界）

### 6.1 总体格局

乱星海是独立于天南大陆的广阔海域，韩立通过古传送阵抵达。

```
乱星海
├── 内星海（已探明海域，星宫统治）
│   ├── 中部：天星城（星宫总部）
│   ├── 内十二星岛（星宫直辖）
│   └── 外二十四星岛（百年一次摘星会选拔岛主）
│       ├── 西南海域：魁星岛、尾星岛、桑星岛
│       ├── 其他星岛
│       └── 各星岛附属数百中小岛屿
├── 外星海（妖海，未探明外围海域）
│   └── 妖兽岛（通过传送阵可达，猎杀妖兽获取妖丹）
└── 其他势力
    ├── 极阴岛（极阴老祖）
    ├── 六连殿
    ├── 四大商盟
    ├── 夜龙岛
    ├── 魅魔宗
    └── 三仙门
```

### 6.2 星宫（乱星海官方势力）

- **总部：** 天星城（建在内星海中部的一个巨岛上）
- **统治者：** 天星双圣（两位城主，从未出城）
- **境界猜测：** 元婴后期乃至化神期
- **内部结构：**
  - 内十二星岛：由双圣各选六人任命
  - 外二十四星岛：每百年"摘星大会"选拔
- **统治特点：** 超然中立，不干涉内部争斗

### 6.3 天星城

- **人口/规模：** 乱星海第一大城
- **核心"圣山"：** 将整座巨山建成城市，盘旋共八十一层
  - 前五十层：对外开放，可租住宅/洞府
  - 后三十层：星宫人员居住
  - 最高一层：圣宫（双圣住所，禁入）
- **灵脉品质：** 乱星海前五，山腰以上可加速修炼四分之一
- **坊市：** 圣山第一层，规模极大占据一半区域
- **特色设施：** 通往外星海的传送阵（可传送至数十个妖兽岛）
- **居住规则：** 蓝色指环（临时/7天）、红色指环（永久，需800灵石）
- **结丹修士：** 可免费永久居住

### 6.4 乱星海势力格局

| 势力 | 性质 | 领袖 |
|------|------|------|
| **星宫** | 官方统治 | 天星双圣 |
| **极阴岛** | 魔道势力 | 极阴老祖（元婴期） |
| **六连殿** | 商业势力 | 六大商家联合 |
| **四大商盟** | 顶级商业联盟 | 未详述 |
| **夜龙岛** | 独立势力 | 未详述 |
| **魅魔宗** | 魔道宗门 | 未详述 |
| **三仙门** | 正道宗门 | 未详述 |

### 6.5 妖兽体系与海域

- **外星海（妖海）：** 未探明的外围海域，强大妖兽横行
- **妖兽等级**（1-10级）：
  - 1-3级 → 炼气/筑基
  - 4-6级 → 结丹（妖丹价值极高）
  - 7-8级 → 元婴
  - 9级 → 化神
  - 10级+ → 炼虚+（化形）
- **妖丹价值：** 5级妖丹≈千灵石，6级妖丹可供筑基修士一生无忧
- **妖兽岛：** 天星城传送阵可达的外星海岛屿，有灵气，可猎杀妖兽

### 6.6 虚天殿（乱星海核心副本）

- **位置：** 乱星海某独立空间
- **开启频率：** 数百年一次
- **进入门槛：** 结丹期以上
- **核心宝物：** 虚天鼎（洪荒古宝）、补天丹
- **主要势力：** 星宫、极阴岛、六连殿、各大散修

### 6.7 魁星岛（韩立落脚第一站）

- **位置：** 乱星海西南角外二十四星岛之一
- **附近岛屿：** 尾星岛、桑星岛，及中小岛屿几十个
- **岛主：** 由摘星大会选拔
- **管理结构：** 登仙阁（办理定居）
- **灵脉：** 主岛灵脉最好，附属岛屿较差
- **魔修入侵：** 韩立曾在魁星岛遭极阴岛通缉而逃离

## 九、设计要点总结

### 9.1 核心设计原则

1. **渐进式地图展开：** 玩家从七玄门（凡人江湖）出发 → 越国七派（低阶修仙） → 乱星海（中阶修仙） → 天南/九国盟（高阶修仙），每一步都自然推进
2. **境界门槛清晰：** 每个区域有明确的境界门槛，低阶玩家无法进入高阶区域
3. **连接通路多样：** 普通路径（步行/马车/飞行）+ 特殊路径（传送阵/秘境入口）
4. **区域特色鲜明：** 每个区域有自己的势力、资源、危险程度和特色玩法

### 9.2 地图展开顺序（按韩立行历）

```
七玄门（凡人江湖） → 越国江湖（岚州） → 太南谷（修仙界初接触）
→ 升仙大会（入黄枫谷） → 血色禁地（筑基试炼）
→ 正魔大战（越国各地） → 古传送阵（出海）
→ 乱星海·魁星岛（结丹） → 天星城（元婴）
→ 外星海（猎妖） → 虚天殿（夺宝）
→ 回天南（九国盟/慕兰边境） → 大晋帝国（后期）
```

### 9.3 进入门槛建议（游戏化）

| 境界段 | 可开放区域 |
|--------|-----------|
| 凡人（无修为） | 镜州江湖（七玄门/野狼帮区域） |
| 炼气初~中期 | 越国全境、太南谷 |
| 炼气后期~筑基 | 血色禁地、乱星海外围岛屿 |
| 结丹期 | 天星城、乱星海各岛、外星海猎妖 |
| 元婴期 | 虚天殿、慕兰草原、九国盟核心区域 |
| 化神期 | 飞升灵界（下一阶段） |
