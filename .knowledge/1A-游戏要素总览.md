## 一、世界观与三界结构

凡人修仙传采用经典的「下界→中界→上界」三阶飞升结构，游戏世界以此为基础展开。

### 1.1 三界总览

| 界域 | 对应境界 | 灵气浓度 | 核心特征 |
|------|---------|---------|---------|
| **人界（凡间界）** | 凡人→化神 | 低 | 修仙者与凡人共存，化神以上被法则排斥 |
| **灵界（中界）** | 化神→大乘 | 极高 | 30+种族并存，面积无限广阔，灵气远超下界 |
| **仙界（上界）** | 真仙+ | 最高 | 故事终点，韩立飞升至北寒仙域，游戏中为终极区域预留 |

**三界通道规则：**
- 化神以上修士被界面排斥，必须飞升灵界（不可逆转）
- 灵界→仙界需渡飞升劫
- 高阶修士降臨低阶位面會被界面法则压制修为

关联文档：[WORLD_DESIGN](WORLD_DESIGN.md)

### 1.2 人界地理格局（韩立成长舞台）

人界故事分为**天南地区**（越国七派、乱星海等）和**大晋帝国**两个核心舞台：

```
天南地区
├── 越国（十三州，七大门派割据，核心舞台）
│   ├── 镜州（七玄门江湖世界）
│   ├── 岚州（嘉元城、太南谷）
│   ├── 建州（黄枫谷、血色禁地）
│   ├── 溪州/辛州/越州等其余八州
├── 天罗国（魔道六宗老巢）
├── 风都国（正道盟大本营）
├── 九国盟（南部九国联盟，抵御慕兰法士）
├── 慕兰大草原（法士领地，尽头通大晋）
├── 大晋帝国（人界第一大国，十倍于天南）
└── 乱星海（海外群岛世界）
    ├── 内星海（星宫统治的天星城）
    └── 外星海（未探明妖兽海域）

凡人江湖 → 越国修仙 → 乱星海 → 天南/九国盟 → 大晋
```

**关键地标：** 七玄门（韩立起点）、太南谷（修仙者坊市）、血色禁地（筑基秘境）、天星城（乱星海核心）、虚天殿（元婴副本）、昆吾山（魔界通道封印地）

关联文档：[1A-人界地理](1A-人界地理.md)

### 1.3 灵界格局

| 大陆/区域 | 主要种族 | 特点 |
|----------|---------|------|
| **雷鸣大陆** | 人族、妖族、角蚩族、海王族 | 灵界核心区域，各族杂居 |
| **血天大陆** | 夜叉族、飞灵族、血族 | 灵界另一大陆板块 |
| **天渊城** | 人族联军 | 抵御魔族的前线要塞 |
| **地渊（多层）** | 远古妖兽 | 地下洞窟世界，越深越危险 |
| **广寒界** | 精英试炼场 | 上古战场遗迹 |

关联文档：[WORLD_DESIGN](WORLD_DESIGN.md)

### 1.4 魔界（异界）

与灵界接壤的异位面，魔族入侵是灵界后期主线。

**魔族等级体系：** 低阶魔族 → 魔将 → 魔帅 → 魔王 → 魔尊 → 圣祖（大乘级）→ 真魔始祖（真仙级）

关联文档：[FACTIONS](FACTIONS.md)

## 二、修仙境界体系（核心成长线）

### 2.1 三阶九境

| 层级 | 境界 | 子阶段 | 寿命 | 特征 |
|------|------|--------|------|------|
| **下境界** | **炼气期** | 1-13层 | ~100年 | 新手阶段，感应灵气打通经脉 |
| (人界) | **筑基期** | 初/中/后期 | ~200年 | 真正踏入修仙界门槛；可施先天真火 |
| | **结丹期** | 初/中/后期 | ~500年 | 战力质变；可炼制法宝、制作符宝 |
| | **元婴期** | 初/中/后期 | ~1000年 | 元婴不灭则不死；法术质变 |
| | **化神期** | 初/中/后期 | ~2000年 | 人界天花板，必须飞升灵界 |
| **中境界** | **炼虚期** | 初/中/后期 | ~5000年 | 化神后吸纳灵界灵气 |
| (灵界) | **合体期** | 初/中/后期 | ~10000年 | 法体合一 |
| | **大乘期** | 初/中/后期 | ~数万年 | 每万年一次大天劫 |
| **上境界** | **真仙** | 金仙/太乙/大罗/道祖 | 理论上永生 | 渡过飞升劫后飞升仙界 |

### 2.2 境界突破概率（基于原文）

| 突破路径 | 基础概率 | 辅助手段 |
|---------|---------|---------|
| 炼气 → 筑基 | ~10%（有筑基丹） | 筑基丹、达第7层以上 |
| 筑基 → 结丹 | ~1% | 雪灵水+天火液、三转重元功 |
| 结丹 → 元婴 | 极低 | 补天丹、心魔劫 |
| 元婴 → 化神 | 几乎不可能（人界内） | 飞升灵界的机缘 |

### 2.3 瓶颈突破方式

1. **丹药辅助**：筑基丹、凝液丹、结金丹、元婴丹、化神丹等
2. **机缘突破**：秘境历练、高人灌顶
3. **战斗突破**：生死一线激发潜力
4. **积累突破**：天灵根专属（无瓶颈自动结丹）
5. **特殊功法**：三转重元功可绕开结丹瓶颈

**心魔与天劫：** 突破大境界时触发心魔；化神飞升/大乘天劫/飞升劫均为雷劫考验。

关联文档：[1C-修仙境界功法](1C-修仙境界功法.md)、[CULTIVATION_SYSTEM](CULTIVATION_SYSTEM.md)

## 三、灵根与功法系统

### 3.1 灵根类型

| 类型 | 属性数 | 修炼速度 | 结丹瓶颈 | 出现概率 |
|------|--------|---------|---------|---------|
| **伪灵根** | 4-5种混杂 | 极慢（x0.3） | 极难 | 绝大多数修仙者 |
| **真灵根** | 2-3种 | 正常（x1.0） | 正常瓶颈 | 有灵根者中~20% |
| **天灵根** | 1种纯属性 | 2-3倍（x2.5） | **无瓶颈自动结丹** | 数百年一遇 |
| **变异灵根** | 雷/冰/风/暗 | 接近天灵根（x2.5） | 正常瓶颈但战力更强 | 每二三十年出现一个 |

**灵根检测与生成：** 一万个人中才可能有一个有灵根者；五六个有灵根者中只能找出一个真灵根；灵根易在相同血脉继承中出现→形成修真家族。

**后天改善：** 补天丹可洗练灵根、洗灵池/净灵莲等特殊天材地宝可提升灵根品质。

### 3.2 功法体系

**按品阶分：**
| 品阶 | 适用境界 | 代表作 |
|------|---------|-------|
| 基础功法 | 炼气-筑基 | 长春功、基础五行诀 |
| 中等功法 | 筑基-结丹 | 青元剑诀、大衍诀、真阳决 |
| 高阶功法 | 结丹-元婴 | 明王诀、托天魔功 |
| 顶级功法 | 化神以上 | 五藏锻元功、炼神术、梵圣真魔功 |

**按系别分：**
- **剑修系**：青元剑诀（韩立核心功法，从结丹用到飞升）
- **法修系**：各类五行法术
- **体修系**：梵圣真魔功（法体双修）、涅槃圣体
- **辅助系**：大衍诀（神识增强）、炼神术（神识分化、控制傀儡）

**核心功法：**
- **长春功**：木系基础，韩立起点，13层大圆满后可尝试筑基
- **青元剑诀**：13层木系剑诀，含大庚剑阵、剑影分光术、无形剑气等神通
- **大衍诀**：4层神识功法，可分神操控多目标
- **梵圣真魔功**：法体双修顶级功法，三头六臂魔神法相
- **炼神术**：顶级神识功法，第三层神识可媲美真仙

### 3.3 五行法术体系

| 属性 | 特点 | 代表性法术 |
|------|------|-----------|
| 金 | 攻击凌厉、锐利 | 金刃术、大庚剑阵 |
| 木 | 生命力、恢复、缠绕 | 缠绕术、回春术、青元剑诀 |
| 水 | 控制、变化、防御 | 冰箭术、冰冻术、乾蓝冰焰 |
| 火 | 爆发、范围伤害 | 火弹术、连珠火球、紫罗天火 |
| 土 | 防御、沉重 | 土墙术、地刺术 |

**相克关系：** 金→木→土→水→火→金（相克循环）

**异火体系：** 乾蓝冰焰（至阴至寒）→ 紫罗天火 → 修罗圣火（冰火雷三属性合一）

关联文档：[1C-修仙境界功法](1C-修仙境界功法.md)、[1F-法术剑诀阵法战斗](1F-法术剑诀阵法战斗.md)、[COMBAT_SYSTEM](COMBAT_SYSTEM.md)

## 四、势力体系（阵营/门派/种族）

### 4.1 人界势力

**越国七大门派：**
| 门派 | 实力 | 特点 |
|------|------|------|
| **掩月宗** | 最强 | 提倡双修，女修为主 |
| **灵兽山** | 第二 | 擅长驱兽役虫（实为魔道卧底） |
| **黄枫谷** | 中等 | 主角韩立所在门派 |
| **清虚门/化刀坞/天阙堡/巨剑门** | 中等 | 各有所长 |

**越国七派关系：** 七派共同把持越国修仙界资源，表面中立合作，内部有利益争夺。魔道入侵时各派联手对抗，但灵兽山临阵倒戈。

**乱星海势力：**
| 势力 | 领袖 | 性质 |
|------|------|------|
| 星宫 | 天星双圣 | 官方统治（超然中立） |
| 逆星盟 | 万天明、六道极圣 | 反星宫联盟 |
| 极阴岛 | 极阴老祖 | 魔道势力 |
| 六连殿/四大商盟 | — | 商业势力 |

**大晋势力：** 正道盟 vs 魔道六宗（鬼灵门、合欢宗、御灵宗等）

### 4.2 灵界种族（30+）

| 种族 | 与主角关系 | 特点 |
|------|-----------|------|
| **人族** | 主角所属 | 修仙体系最完善 |
| **妖族** | 盟友（银月/敖啸） | 妖兽化形，肉身强悍 |
| **木族** | 附庸/被保护 | 植物形态种族 |
| **角蚩族** | 敌对 | 好战的灵界强族 |
| **海王族** | 竞争 | 海洋种族 |
| **夜叉族/飞灵族/空鱼族/晶族等** | 中立/归附 | 各具特色 |

### 4.3 魔界势力

| 层级 | 描述 |
|------|------|
| 真魔始祖 | 魔界最顶层（堪比真仙） |
| 圣祖 | 大乘级别魔族（元魇、宝花等） |
| 魔尊 | 合体级别 |
| 普通魔族 | 基层战力 |

### 4.4 势力关系图谱

**人界：**
```
魔道六宗 ←→ 正道盟（大晋对峙）
          ↘
  越国七派（正魔不明显，利益联盟为主）
     ↓
  乱星海：星宫 vs 逆星盟
```

**灵界：**
```
     [魔界] —（入侵）→ [灵界各族]
人族 ←→ 妖族（同盟） → 木族（附庸）
  ↓
角蚩族/海王族（竞争者）
夜叉/飞灵等（中立种族）
```

**声望系统：** 每个势力有独立声望（陌生→友善→信任→尊敬→崇拜），影响购买、任务、拜师、功法等。

## 六、战斗系统

### 6.1 境界与战斗力

每差一个大境界战力差距极大，但顶级装备/功法可跨级击杀（韩立多次以低境界击杀高境界敌手）。

### 6.2 五大子系统

| 系统 | 核心内容 |
|------|---------|
| **法术** | 五行法术（金/木/水/火/土）+ 变异（雷/冰/风/暗） |
| **剑诀** | 以青元剑诀为核心，飞剑养成→剑阵组合→终极形态 |
| **体修** | 梵圣真魔功（三头六臂魔神法相）等近战功法 |
| **阵法** | 防御阵/攻击阵/困阵/传送阵/封印阵，颠倒五行阵为代表 |
| **傀儡/灵兽** | 低阶→真仙级傀儡；噬金虫、银翅夜鹏等灵兽 |

### 6.3 特殊战斗机制

| 机制 | 说明 |
|------|------|
| **夺舍** | 高境界→低境界施展，一生一次，失败则元神反噬 |
| **心魔** | 突破大境界时触发，强度与业力相关 |
| **天劫** | 化神飞升（小天劫）、大乘万年（大天劫）、飞升（终极劫） |
| **身外化身** | 筑基+大衍诀2层后可炼制第二身体 |
| **血影遁** | 以精血为代价的逃生瞬移 |

## 八、角色体系

### 8.1 主角韩立

- **出身**：越国青牛镇穷山村小子
- **灵根**：四属性伪灵根（前期极慢，靠掌天瓶+丹药提速）
- **性格**：谨慎、低调、心思缜密、该狠则狠
- **绰号**：二愣子、韩跑跑、韩老魔
- **修仙里程碑**：七玄门→黄枫谷→乱星海→元婴→飞升灵界→大乘→飞升仙界

### 8.2 重要角色分类

| 类别 | 代表角色 |
|------|---------|
| **亲友/道侣** | 南宫婉（妻/掩月宗）、元瑶、银月/玲珑、紫灵 |

## 十、设计原则汇总

1. **渐进式体验**：玩家从凡人江湖出发→越国修仙→乱星海→灵界，每步自然推进
2. **境界为核心**：境界是等级、战力、区域准入、任务门槛的统一标准
3. **灵根定流派**：灵根决定可学功法和战斗力上限，影响玩家成长路线选择
4. **寿命做驱动力**：寿命倒计时是修仙游戏区别于传统RPG的核心机制
5. **法宝可成长**：核心法宝（如青竹蜂云剑、掌天瓶）可随玩家成长进化
6. **声望开门槛**：势力声望解锁专属任务/功法/区域，驱动玩家探索
